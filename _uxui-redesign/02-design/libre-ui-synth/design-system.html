마케팅 분석 에이전트가 월간 리포트 47건을 성공적으로 완료했습니다.

토큰 만료 임박

영업 지원 에이전트의 CLI 토큰이 3일 후 만료됩니다. 미리 갱신하세요.

연결 오류

데이터 수집 에이전트가 외부 API 연결에 실패했습니다. 자격 증명을 확인하세요.

시스템 공지

오늘 오전 2시~4시 사이 정기 점검이 예정되어 있습니다. 에이전트 작업이 일시 중단될 수 있습니다.
