Tool Schema Registry

Master list of all executable functions (tools) that can be bound to agents. Describes JSON schemas, required clearances, and execution environments.

core_search

T-0

Execute semantic vector search over the tenant's RAG knowledge base. Allows bounding by namespace array.

IN: { "query": string, "namespaces": string[] }
OUT: { "results": Document[] }
Usage: 142k / mo

delegate_task

ROOT ONLY

Spawns a child agent thread with a specific context and task. Only Secretary General has native clearance.

IN: { "target_agent": id, "task": string }
OUT: { "completion_status": enum, "result": string }
Usage: 2.1M / mo

fetch_kis_account

T-1

Wrapper for Korea Investment & Securities API. Returns current portfolio holdings and cash balance. Requires tenant to have valid KIS credentials mapped.

IN: {}
OUT: { "holdings": Asset[], "cash": USD }
Usage: 840 / mo
