시스템 도구(Tools) 레지스트리

에이전트들이 사용할 수 있는 전역 함수 집합(Function Calling)과 외부 API 연동 명세를 관리합니다.

인터넷 브라우저 / 검색

tool_web_search_v2

내부 코어

에이전트가 실시간 웹 검색 및 페이지 스크래핑을 수행할 수 있는 권한을 제공합니다. (Tavily Search API 기반)

오픈 API 명세 (일부) 
{
  "name": "search_web",
  "description": "Searches the web for recent info.",
  "parameters": {
    "type": "object",
    "properties": {
      "query": { "type": "string" }
    }
  }
}
허가된 직무 템플릿: 전체(All)

PostgreSQL 질의기

tool_sql_executor

고객사 인프라 연동

고객사가 연동한 데이터베이스에 안전한 Read-Only SQL 쿼리를 실행하고 결과를 데이터프레임으로 변환합니다.

오픈 API 명세 (일부) 
{
  "name": "execute_query",
  "description": "Runs a SQL SELECT query.",
  "parameters": {
    "type": "object",
    "properties": {
      "sql": { "type": "string" }
    }
  }
}
허가된 직무 템플릿: 데이터 분석가, CFO
