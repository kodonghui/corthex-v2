- Nano Banana 2로 이미지 시안을 먼저 만들고 → HTML로 변환
- Claude보다 토큰 비용이 저렴 → 대량 생성에 유리

## 컨셉별 디자인 방향 예시 (5가지)

### 컨셉 1: 네오 브루탈리즘 (Neo-Brutalism)
**레퍼런스**: Gumroad, Figma Community, Notion Calendar
**상세 설명**:
```
- 굵은 검은 테두리(2~4px solid black), 강한 드롭 섀도우(offset, 블러 없음)
- 밝고 과감한 원색 배경 (노랑, 라임, 핫핑크, 하늘색을 블록처럼 사용)
- 큼직한 산세리프 타이포그래피, 볼드 헤딩
- 버튼은 3D 느낌 (눌리는 효과), 카드는 두꺼운 보더 + 컬러 배경
- 재미있고 에너지 넘치는 느낌. 뻔한 SaaS 느낌 절대 내지 마.
```

### 컨셉 2: 글래스모피즘 다크 (Glassmorphism Dark)
**레퍼런스**: Apple Vision Pro UI, Linear, Arc Browser
**상세 설명**:
```
- 어두운 배경(#0a0a0f ~ #1a1a2e) 위에 반투명 글래스 카드 (backdrop-blur, bg-white/5~10%)
- 미묘한 보더(border-white/10%), 부드러운 글로우 효과
- 액센트 컬러는 보라~시안 그라데이션 (purple-500 → cyan-400)
- 깔끔한 산세리프 폰트, 얇은 폰트웨이트 + 볼드 숫자
- 고급스럽고 미래적인 느낌. 우주/공상과학이 아니라 프리미엄 프로덕트 느낌.
```

### 컨셉 3: 내추럴 오가닉 (Natural Organic)
**레퍼런스**: Notion, Things 3, Bear App
**상세 설명**:
```
- 따뜻한 베이지/크림 배경 (#faf8f5, #f5f0eb), 오프화이트 카드
- 액센트는 테라코타(#c4622d), 올리브그린(#5a7247), 머스타드(#d4a843)
- 세리프 + 산세리프 혼합 타이포 (헤딩은 세리프, 본문은 산세리프)
- 부드러운 그림자, 넉넉한 패딩, 둥근 모서리(12~16px)
- 편안하고 따뜻한 느낌. 차가운 테크 느낌 절대 내지 마.
```

### 컨셉 4: 모노크롬 타이포그래피 (Monochrome Typography)
**레퍼런스**: Stripe, Apple Developer, Rauno.me
**상세 설명**:
```
- 순수 흑백 기반 (#000, #111, #666, #999, #eee, #fff). 컬러는 딱 하나만 — 상태 표시용 레드/그린
- 타이포그래피가 주인공. 폰트 사이즈, 웨이트, 자간으로 위계 표현
- 극도의 미니멀. 아이콘 최소화, 텍스트 중심, 여백 과감하게
- 보더/구분선 대신 여백과 타이포 크기로 섹션 구분
- 세련되고 지적인 느낌. 귀엽거나 화려한 거 절대 하지 마.
```

Gemini가 "파일에 접근할 수 없습니다" 하면 절대경로를 직접 알려줘:

```

{워크트리경로}\_uxui-redesign\01-spec\prd.md
{워크트리경로}\_uxui-redesign\01-spec\api-endpoints.md
{워크트리경로}\_uxui-redesign\01-spec\shared-types.ts

결과물 저장 경로:
{워크트리경로}\_uxui-redesign\02-design\{도구이름}\

이 경로들 읽고 작업 시작해.
```

## 색상 변형 뽑기

마음에 드는 디자인이 하나 나오면, 색상만 다르게 변형 가능:

```
방금 만든 디자인 구조는 그대로 유지하고, 색상 테마만 바꿔서 3가지 버전을 만들어줘:
1. 다크 테마 버전 (배경 어둡게)
2. 라이트 테마 버전 (배경 밝게)
3. 다른 액센트 컬러 버전 ({원하는 색상})
레이아웃, 폰트, 간격은 전부 동일하게. CSS 변수(색상)만 바꿔.
```
