Chrome을 직접 열어서 실행하므로 화면으로 볼 수 있습니다.

## 프롬프트 (복사해서 붙여넣기)

```
당신은 "사주포춘"이라는 AI 운세 상담 사이트를 운영할 회사를 CORTHEX 플랫폼에서 세팅하고 실제로 운영해보는 테스트를 합니다.

Chrome 브라우저를 열어서 아래 단계를 순서대로 실행하세요. 각 단계마다 화면을 잠시 멈춰서 결과를 확인한 후 다음으로 넘어가세요.

## PART 1: Admin 패널에서 회사 세팅 (15분)

### 1-1. 로그인
1. Chrome에서 https://corthex-hq.com/admin/login 접속
2. 관리자 ID: admin 입력
3. 비밀번호: admin1234 입력
4. "세션 시작" 버튼 클릭
5. 대시보드가 나오면 성공

### 1-2. 온보딩 (사이드바 → 온보딩 클릭)
1. 왼쪽 사이드바에서 "온보딩" 메뉴 찾아서 클릭 (또는 URL: /admin/onboarding)
2. Step 1 (회사): 회사 이름이 "CORTHEX HQ"로 되어있을 텐데, 클릭해서 "사주포춘"으로 변경 → 저장
3. Step 2 (조직): "올인원" 템플릿을 찾아서 "적용" 클릭. 실패하면 "빈 조직으로 시작" 클릭
4. Step 3 (API 키): API 키 입력란이 있으면 Anthropic 키 입력. 없으면 "건너뛰기" 클릭
5. Step 4 (팀원 초대): 이름 "사주포춘 대표", 아이디 "sajuceo" 입력 → 초대하기. 임시 비밀번호 기억
6. Step 5 (요약): 확인 후 "CORTHEX 사용 시작하기" 클릭

### 1-3. 부서 생성 (사이드바 → 조직 관리 → 부서 관리)
1. /admin/departments 페이지로 이동
2. "Create Department" 버튼 클릭
3. 순서대로 4개 부서 생성:
   - 이름: "운세분석팀", 설명: "사주팔자, 타로, 별자리 분석 전문"
   - 이름: "마케팅팀", 설명: "SNS, 블로그, 콘텐츠 마케팅"
   - 이름: "고객서비스팀", 설명: "고객 문의 응대, FAQ 관리"
   - 이름: "경영지원팀", 설명: "재무, 법무, 사업기획"

### 1-4. AI 에이전트 생성 (사이드바 → AI 관리 → AI 에이전트)
1. /admin/agents 페이지로 이동
2. "NEW AGENT" 또는 "에이전트 생성" 버튼 클릭
3. 아래 6개 에이전트를 순서대로 생성:

**에이전트 1: 운세분석관**
- 이름: 운세분석관
- 역할: 사주/타로 분석 전문가
- 티어: Specialist
- 모델: Claude Haiku 4.5
- 부서: 운세분석팀
- 소울 (길게 적기): "당신은 사주포춘의 수석 운세 분석관입니다. 동양 명리학(사주팔자, 음양오행), 서양 점성술(12궁 별자리), 타로 카드 해석에 깊은 전문성을 보유하고 있습니다. 분석은 항상 정확하고 구체적으로, 톤은 신비로우면서도 따뜻하게 전달합니다. 모든 답변은 한국어 존댓말로 작성합니다."

**에이전트 2: 마케팅매니저**

- 역할: 디지털 마케팅 전략 수립
- 티어: Manager

- 소울: "당신은 사주포춘의 CMO(마케팅 총괄)입니다. MZ세대를 주요 타겟으로 인스타그램, 블로그, 유튜브 숏츠 콘텐츠 전략을 수립합니다. 트렌디하고 감각적인 카피라이팅이 특기이며, 데이터 기반 마케팅 의사결정을 합니다. 콘텐츠 캘린더, 해시태그 전략, 타겟 페르소나 분석을 체계적으로 수행합니다."

### 1-5. API 키 등록 (사이드바 → 운영 → CLI / API 키)
1. /admin/credentials 페이지로 이동
2. 사용자를 "관리자"로 선택
3. API 키 섹션에서:
   - 프로바이더: "anthropic" 선택
   - 라벨: "Claude API"
   - API Key 필드에 위의 Anthropic 키 붙여넣기
   - "등록" 클릭
4. 한번 더:
   - 프로바이더: "serper" 선택
   - 라벨: "Serper 웹검색"
   - API Key 필드에 위의 Serper 키 붙여넣기
   - "등록" 클릭

### 1-6. 에이전트 온라인으로 변경
1. /admin/agents 페이지로 돌아가기
2. 각 에이전트를 클릭해서 상세 패널 열기
3. 상태를 "online"으로 변경할 수 있는 방법 찾기 (편집 버튼 → status 필드)
4. 최소 비서실장 + 운세분석관 + 마케팅매니저를 online으로 변경

## PART 2: CEO App에서 실제 업무 (20분)

### 2-1. App 로그인
1. 새 탭에서 https://corthex-hq.com/login 접속
2. 아이디: admin, 비밀번호: admin1234 입력 → 로그인
3. Hub 페이지가 나오면 성공

### 2-2. Hub 둘러보기
1. Hub 화면 구성 확인: 왼쪽 사이드바, 중앙 채팅 영역, 예시 명령어 버튼
2. 예시 명령어 버튼이 있으면 하나 클릭해보기
3. 좌측 사이드바에서 메뉴 항목들 확인

### 2-3. 비서실장과 대화 (핵심 테스트!)
1. /chat 페이지로 이동 (사이드바에서 "채팅" 클릭)
2. "새 대화" 또는 "+" 버튼 클릭
3. 에이전트 목록에서 "비서실장" 선택
4. 대화창에 아래 메시지를 입력하고 전송:

첫 번째 지시:
```
사주포춘 인스타그램 마케팅 전략 기획서를 써줘.
타겟: MZ세대 (20-35세)
포함 내용: 브랜드 포지셔닝, 콘텐츠 유형별 전략, 1개월 콘텐츠 캘린더, 해시태그 전략
```

5. 응답이 올 때까지 기다리기 (30초~2분 소요)
6. 응답 품질 확인: 실제로 쓸 만한 기획서가 나왔는지

### 2-4. 운세분석관과 직접 대화
1. 새 대화 시작 → "운세분석관" 선택
2. 메시지 입력:
```
1995년 3월 15일생 여성의 2026년 종합 운세를 봐줘.
사주팔자 기반으로 해줘. 연애운, 재물운, 건강운, 직장운 각각 분석해줘.
```
3. 응답 대기 → 품질 확인

### 2-5. 마케팅매니저와 대화
1. 새 대화 → "마케팅매니저" 선택
2. 메시지:
```
인스타 게시물 5개 써줘. 주제는 사주/타로 관련.
각 게시물마다: 제목, 본문 (150자 이내), 해시태그 10개 포함.
```
3. 응답 대기 → 품질 확인

### 2-6. 기획담당과 대화
1. 새 대화 → "기획담당" 선택
2. 메시지:
```
사주포춘 사업계획서를 써줘.
포함: 시장 분석(온라인 운세 시장 규모), 경쟁사 분석(점신, 신한점, 운세박사 등),
수익모델(구독/건당/프리미엄), 1년 매출 목표, 필요 투자금
```
3. 응답 대기 → 품질 확인

### 2-7. 법무담당과 대화
1. 새 대화 → "법무담당" 선택
2. 메시지:
```
온라인 사주/타로 상담 서비스 이용약관 초안을 써줘.
전자상거래법, 개인정보보호법 준수. 환불 정책, 면책 조항 포함.
```
3. 응답 대기 → 품질 확인

## PART 3: 기타 기능 테스트 (10분)

### 3-1. 작업 관리 (/jobs)
1. 사이드바에서 "작업" 클릭
2. "새 작업" 또는 "+" 버튼 클릭
3. 작업명: "매일 오늘의 운세 생성"
4. 에이전트: 운세분석관 선택
5. 지시: "오늘의 12궁 별자리 운세를 작성해줘. 각 별자리 3줄 이내."
6. 저장 또는 실행

### 3-2. 지식 베이스 (/knowledge)
1. 사이드바에서 "지식" 클릭
2. "새 폴더" 생성: "사주 이론"
3. 폴더 안에 "새 문서" 생성:
   - 제목: "사주 기초 이론"
   - 내용: "사주팔자란 태어난 년, 월, 일, 시의 네 기둥(四柱)으로 구성된 여덟 글자(八字)를 의미합니다. 음양오행의 상호작용을 통해 개인의 운명과 성격을 분석합니다."
4. 저장

### 3-3. 에이전트 목록 (/agents)
1. 사이드바에서 "에이전트" 클릭
2. 에이전트 카드들 확인 — 방금 만든 에이전트들이 보이는지
3. 하나 클릭해서 상세 정보 확인

### 3-4. 부서 (/departments)
1. 사이드바에서 "부서" 클릭
2. 생성한 4개 부서 확인
3. 하나 클릭해서 소속 에이전트 목록 확인

### 3-5. 비용 (/costs)
1. 사이드바에서 "비용" 클릭
2. 채팅에서 사용한 비용이 기록되어 있는지 확인

### 3-6. 알림 (/notifications)
1. 사이드바에서 "알림" 클릭
2. 알림이 있는지 확인

### 3-7. 설정 (/settings)
1. 사이드바에서 "설정" 클릭
2. 각 탭 둘러보기 (일반, 테마, 소울편집 등)
```
