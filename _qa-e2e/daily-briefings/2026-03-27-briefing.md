# 클로드코드 데일리 브리핑 — 2026-03-27

## 1. Claude Code 업데이트

- **현재 최신 버전**: 2.1.84 (2026-03-26 출시)
- **주요 신기능**:
  - PowerShell 툴 (Windows opt-in 프리뷰)
  - `TaskCreated` 훅 — TaskCreate로 작업 생성 시 발동
  - `WorktreeCreate` 훅에 `type: "http"` 지원 추가
  - `CLAUDE_STREAM_IDLE_TIMEOUT_MS` 환경변수 (스트리밍 idle watchdog, 기본 90초)
  - `/clear` 시 캐시된 스킬 초기화 버그 수정
- **/dream 공식 상태**: **미출시** (공식 changelog에 언급 없음)
- **3월 주요 변경사항**:
  - **음성 모드** (`/voice`): spacebar push-to-talk 방식으로 점진적 롤아웃 중
  - **`/loop` 명령어**: 주기적 프롬프트 실행 (예: `/loop 5m check deploy`)
  - **1M 토큰 컨텍스트**: v2.1.75(3/13)부터 Opus 4.6 기본 적용 (Max/Team/Enterprise)
  - **Opus 4.6 기본 모델 전환**: Opus 4/4.1 API 제거, 4.6으로 강제 마이그레이션
  - **`ultrathink` 키워드 부활**: 다음 턴 "high" effort 활성화
