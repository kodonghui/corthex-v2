## 1. Claude Code 업데이트

- **현재 최신 버전**: v2.1.59+ (정확한 버전은 `claude --version` 확인)
- **새 기능** (3월 23일 업데이트):
  - Computer Use 추가 (Pro/Max): 클릭, 파일 열기, 개발 도구 실행 — 별도 설정 불필요
  - Scheduled Tasks: 컴퓨터 꺼진 상태에서도 Anthropic 클라우드에서 반복 작업 실행
  - PreToolUse hook에 "defer" 권한 추가 — 헤드리스 세션에서 tool call 일시정지 후 재개
  - `/powerup` 명령어: 애니메이션 데모로 기능 학습
  - Claude Opus 4.6 기본 최대 출력 토큰 64k → 상한 128k
  - Write 도구 diff 계산 60% 속도 향상
- **Claude Code Security**: 코드베이스 보안 취약점 자동 검토 기능 출시 (2월)
- **/dream 공식 상태**: **아직 미출시** — feature flag로 일부 제한, GitHub 이슈 #38461/#39135에서 버그 보고 중. 우리 `/dream`은 커스텀 구현이므로 영향 없음.
