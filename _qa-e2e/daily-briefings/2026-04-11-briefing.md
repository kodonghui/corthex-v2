  - **`/cost` 상세화**: 모델별·캐시히트별 비용 분류 (구독 유저)
  - **Write 도구 속도**: 대용량 파일 diff 계산 60% 빠르게
- **`/dream` 공식 상태**: **아직 미출시 (조용한 롤아웃 중)**
  - v2.1.59+부터 바이너리에 포함되어 있으나 전 유저 미지원
  - 공식 문서·블로그 포스트 없음
  - Auto-Dream: 세션 간 메모리 파일 정리·통합 (REM 수면 비유)
  - 수동 트리거: Claude에게 "dream" / "consolidate my memory files" 라고 말하면 됨
