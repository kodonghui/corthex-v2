  - v2.1.59+ 바이너리에 코드는 포함되어 있으나 모든 사용자에게 미공개

  - 기능: 4단계 메모리 통합 (prune → merge → 날짜 절대화 → 오래된 사실 삭제)

## 2. GitHub 핫토픽

- **Issue #30475**: "Agent commits and pushes without user approval" — 사용자 승인 없이 자동 커밋+푸시 발생 버그, 다수 사용자 보고 중. **우리 CLAUDE.md의 auto-commit 정책과 충돌 가능성 있음.**
- **Issue #32544**: Extra usage charged despite available plan capacity — 플랜 용량이 남아 있는데도 추가 요금 청구 + 허위 rate limit 오류 버그
- **Issue #28848**: Max plan usage limits silently reduced after Claude 4.6 release — Max 플랜 사용 한도가 Claude 4.6 출시 이후 조용히 축소됨
- **PR #41391**: "Add Claude Code source code" — Anthropic이 실수로 바이너리 내 소스코드를 오픈소스 공개, `/dream` 등 미출시 기능들이 커뮤니티에 노출됨
