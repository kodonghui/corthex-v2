**모든 페이지에서 아래를 전수 테스트:**

```
1. 페이지 로드 → 콘텐츠 보이는지 (빈 화면 아닌지)
2. 레퍼런스 screen.png와 비교 → 레이아웃/색상 차이 기록
3. 모든 버튼 → 클릭 → 반응 기록
4. 모든 입력 필드 → 텍스트 입력 → 입력 가능한지
5. 모든 폼 → 입력 후 제출 → 성공/실패/에러 메시지
6. 모든 드롭다운/셀렉트 → 열기 → 옵션 선택 → 변경 확인
7. 모든 탭/토글 → 클릭 → 상태 변경 확인
8. 모든 삭제 → 확인 다이얼로그 → 삭제 반영
9. 모든 검색/필터 → 입력 → 결과 변경 확인
10. CRUD 가능한 곳 → 생성→목록에 나타남→수정→반영→삭제→사라짐
11. 빈 데이터 → empty state 메시지 확인
12. 콘솔 에러 전수 수집
```

## 규칙
- **PRD를 반드시 먼저 읽고** 각 FR을 이해한 후 테스트
- **코드 수정 금지.** 검사 + 기록만.
- **git commit/push 금지.**
- 스크린샷만 찍고 끝내면 **해당 페이지 미검사** 처리
- 각 페이지의 **모든 인터랙션 가능 요소**를 전부 테스트
- Phase 3+ FR이 미구현이면 "미구현"으로 기록 (버그 아님)
- 콘솔 에러 전문 복사 (요약 금지)
