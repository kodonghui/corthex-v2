## ROUND 1: Admin 계정

### 로그인 E2E

1. https://corthex-hq.com 접속
2. **빈 폼 제출** → 에러 메시지 나오는지?
3. **틀린 비번** admin / wrongpass → 에러 메시지 나오는지? 뭐라고?
4. **정상 로그인** admin / admin1234 → 어디로 이동?
5. 로그인 후 사이드바 메뉴 전부 기록 (위→아래)

### /hub — 허브 (핵심 기능 전수)

PRD 기준: "사용자가 허브에서 에이전트에게 자연어로 명령을 보낼 수 있다" (FR1)

**조작 순서대로 전부 테스트:**
1. 채팅 입력창 클릭 → 커서 깜빡이는지?
2. "안녕하세요" 입력 → 글자가 입력되는지?
3. 전송 버튼 클릭 → **실제로 응답이 오는지?** (이게 핵심)
   - 응답이 실시간으로 한 글자씩 나오는지? (FR6 SSE)
   - 아니면 한번에 뿅 나오는지?
   - 아니면 아무 반응 없는지?
   - 에러 메시지가 나오는지?
4. 응답이 왔다면 → 복사 버튼 있는지? 클릭하면 복사되는지? (FR67)
5. 응답에 마크다운(표, 코드블록) → 제대로 렌더링되는지? (FR68)
6. 사이드바에 이전 대화 기록 → 클릭하면 로드되는지? (FR62)
7. 새 대화 버튼 → 빈 화면 초기화되는지?
8. 파일 첨부 버튼 → 파일 선택 가능한지? (FR65)
9. 핸드오프 발생 시 → 트래커에 "○○가 작업 중" 표시되는지? (FR46)
10. 에이전트 선택 UI → 다른 에이전트 선택 가능한지? (FR13)
11. 레퍼런스 이미지와 비교 → 차이점 기록

### /agents — 에이전트 CRUD 전체 사이클

PRD 기준: "Admin이 에이전트를 생성/수정/삭제하고 부서에 배치할 수 있다" (FR27)

1. 목록이 보이는지? 몇 개?
2. **생성**: 추가 버튼 → 폼 열리는지? → 이름 "테스트봇" 입력 → 부서 선택 가능? → 저장 → 목록에 나타나는지?
3. **Soul 편집**: 테스트봇 클릭 → Soul 편집 버튼 → 텍스트 수정 가능? → 저장 → 성공 메시지? (FR21)
4. **수정**: 테스트봇 편집 → 이름 "테스트봇2"로 변경 → 저장 → 반영?
5. **도구 할당**: 테스트봇2 편집 → 도구 할당 UI → 체크/해제 가능? (FR29)
6. **삭제**: 테스트봇2 삭제 → 확인 다이얼로그? → 삭제 후 목록에서 사라지는지?
7. **비서실장 삭제 시도**: 비서실장 삭제 → 시스템이 거부하는지? (FR33)
8. 검색창 → "비서" 입력 → 필터링 동작?

### /departments — 부서 CRUD (FR26)

1. 목록 보이는지? (경영지원실, 개발팀, 마케팅팀, 재무팀)
2. **생성**: 추가 → "QA팀" → 저장 → 목록에 추가?
3. **수정**: QA팀 편집 → "품질관리팀" → 저장 → 반영?
4. **삭제**: 품질관리팀 삭제 → 확인 → 사라짐?
5. 부서 클릭 → 소속 에이전트 보이는지?

### /tiers — 티어 (FR34~FR36)

1. 티어 목록 보이는지?
2. 각 티어에 모델명 표시?
3. 티어 추가/수정/삭제 가능? (Phase 3 — 미구현이면 기록)
4. 에이전트의 티어 변경 가능?

### /jobs — 자동화 (FR55)

1. 잡 목록 보이는지?
2. 상태(실행중/완료/실패) 배지 있는지?
3. 잡 생성 → 이름/스케줄 입력 → 저장 → 목록에 추가?
4. 잡 클릭 → 상세/실행 이력?
5. 필터 동작?
6. 잡 삭제 → 사라짐?

### /nexus — 조직도 (FR30~FR32)

1. React Flow 캔버스 보이는지?
2. 노드(부서/에이전트)가 보이는지?
3. 드래그 이동 가능?
4. 줌인/줌아웃?
5. Admin이면 편집 가능? (FR30, Phase 3)
6. 레퍼런스 비교

### /settings — 설정

1. 탭 몇 개? 이름?
2. 각 탭 클릭 → 내용 바뀌는지?
3. 각 탭에 내용이 있는지? (비어있으면 기록)
4. 저장 버튼 → 클릭 → 반응?
5. 프로필 편집 가능?

### /admin/users — 사용자 (FR28)

1. 사용자 테이블 보이는지? admin/ceo 있는지?
2. 추가 → 폼 → 저장 → 목록 추가?
3. 편집 → 역할 변경 → 저장 → 반영?
4. 삭제 → 확인 → 사라짐?
5. 검색 동작?

### /admin/credentials — 자격증명 (FR43)

1. **API 키 마스킹 확인** (전문 노출 = 보안 버그)
2. 추가 → 저장 → 마스킹 상태?
3. 삭제 → 사라짐?
