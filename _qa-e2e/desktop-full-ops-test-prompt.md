### 1-1. 로그인 (1분)
1. Chrome에서 https://corthex-hq.com/admin/login 접속
2. 관리자 ID: admin, 비밀번호: admin1234 입력

### 1-2. 온보딩 (3분)
왼쪽 사이드바 → "온보딩" 클릭 (또는 /admin/onboarding)

**Step 1 — 회사 설정:**
- 회사명이 "CORTHEX HQ"로 되어있음
- "Edit" 클릭 → "테스트컴퍼니" 입력 → "Save" 클릭
- 토스트 메시지 "회사명이 수정되었습니다" 확인

**Step 2 — 부서 설정:**
- 조직 템플릿 목록이 보임 (투자분석, 마케팅, 올인원 등)
- "올인원" 템플릿의 "Apply" 클릭 → 에이전트 자동 생성되는지 확인
  - 만약 에러 나면: "빈 조직으로 시작" 클릭
- 커스텀 부서 추가: "영업팀" 입력 → "Add" 클릭 → 부서 추가 확인
- 빈 이름으로 "Add" 클릭 → 아무것도 안 되는지 확인

**Step 3 — API 키:**
- Anthropic API 키 입력란에 키 입력 → "등록" 클릭
- "등록됨" 표시 확인
- "Continue" 클릭

**Step 4 — 팀원 초대:**
- 아이디: "testuser", 이름: "테스트유저", 이메일: "test@test.com" 입력
- "초대하기" 클릭 → 임시 비밀번호가 표시되는지 확인
- "Copy" 버튼 → 복사되는지 확인
- "Skip for now" 클릭

**Step 5 — 완료:**
- 요약 정보 확인 (회사명, 템플릿, API 키, 초대 직원 수)
- "CORTHEX 사용 시작하기" 클릭 → 대시보드로 이동

**다른 에이전트:**

9. 입력: "기술 부채 분석 보고서 써줘"
10. 응답 확인

**세션 관리:**
11. 세션 목록에서 이전 세션 클릭 → 대화 이력 로드
12. 세션 이름 변경 가능한지 확인
