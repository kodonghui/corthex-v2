# AI 문체 탈피 상세 가이드 (Anti-AI Writing Patterns)

이 문서는 자소서 작성 시 AI가 작성한 것처럼 보이는 패턴을 감지하고 수정하는 상세 지침이다.
인사담당자가 AI 탐지 도구를 사용하거나 직접 판별하는 경우가 늘고 있으므로, 이 가이드를 반드시 적용한다.

---

## 1. 금지 어휘 목록 (Korean AI Vocabulary Watchlist)

### 과장 수식어 (즉시 교체)
| 금지 표현 | 대체 방향 |
|---|---|
| 혁신적인 | 구체적으로 무엇이 새로웠는지 서술 |
| 획기적인 | 기존 대비 어떻게 달라졌는지 수치로 |
| 탁월한 | 구체적 성과나 평가로 대체 |
| 괄목할 만한 | 실제 결과를 직접 서술 |
| 뛰어난 역량 | 어떤 역량인지 구체적으로 |
| 핵심적인 역할 | 실제로 무엇을 했는지 |

### 공허한 부사 (구체적 행동으로 교체)
| 금지 표현 | 대체 방향 |
|---|---|
| 적극적으로 | "주 3회 고객 미팅을 주도하며" 같은 구체적 행동 |
| 체계적으로 | "주간 보고 체계를 도입하여" 같은 구체적 방법 |
| 열정적으로 | 열정을 보여주는 행동을 서술 |
| 능동적으로 | 어떤 주도적 행동을 했는지 |
| 효율적으로 | 얼마나 효율이 올랐는지 수치로 |
| 창의적으로 | 어떤 아이디어를 냈는지 구체적으로 |

### 클리셰 표현 (완전 재구성)
| 금지 표현 | 문제점 |
|---|---|
| "어렸을 때부터 ~에 관심이 있었습니다" | 천편일률적 도입부 |
| "최선을 다하겠습니다" | 누구나 하는 말, 차별화 불가 |
| "귀사의 비전에 공감하여" | 막연한 회사 칭찬 |
| "4차 산업혁명 시대에" | 시대적 클리셰 |
| "글로벌 경쟁력을 갖춘" | 의미 없는 수식 |
| "~의 중요성을 깨달았습니다" | 깨달음 대신 행동 변화를 서술 |
| "이러한 경험을 바탕으로" | 과다 사용되는 연결어 |
| "한 단계 더 성장할 수 있었습니다" | 어떻게 성장했는지 구체적으로 |

## 2. 문장 구조 패턴 감지

### Pattern 1: 균일한 문장 길이
- 문제: 모든 문장이 20~25자로 비슷한 길이
- 해결: 10자 미만의 짧은 강조 문장과 40자 이상의 설명 문장을 섞는다
- 예시 (나쁨): "데이터를 분석했습니다. 인사이트를 도출했습니다. 보고서를 작성했습니다."
- 예시 (좋음): "2주간의 데이터 분석 끝에, 고객 이탈의 핵심 원인을 찾았습니다. 온보딩 프로세스였습니다."

### Pattern 2: 나열식 서술
- 문제: "~했으며, ~했고, ~했습니다" 반복
- 해결: 인과관계, 시간 순서, 대비 등으로 문장 관계를 다양화
- 예시 (나쁨): "시장 조사를 했으며, 경쟁사를 분석했고, 전략을 수립했습니다."
- 예시 (좋음): "시장 조사에서 발견한 경쟁사의 약점이 전략의 출발점이 되었습니다."

### Pattern 3: "저는" 반복
- 문제: 연속 3문장 이상 "저는"으로 시작
- 해결: 주어 생략, 경험/상황 주어, 다양한 도입부 사용
- 예시 (나쁨): "저는 팀장을 맡았습니다. 저는 회의를 주도했습니다. 저는 결과를 보고했습니다."
- 예시 (좋음): "팀장을 맡으면서 주간 회의를 직접 설계했고, 이 자리에서 나온 아이디어가 최종 결과물의 뼈대가 되었습니다."

### Pattern 4: 접속부사 과다
- 문제: "또한", "더불어", "나아가", "뿐만 아니라" 과다 사용
- 해결: 문장 순서 자체로 논리 흐름을 만들고, 접속부사는 문단당 1회 이하
- 전체 자소서에서 "또한"은 최대 2회, "나아가"는 최대 1회

### Pattern 5: 감정/생각 부재
- 문제: 팩트만 나열하고 개인의 감정이나 판단이 없음
- 해결: "~라고 판단했습니다", "처음엔 막막했지만", "이때 느낀 보람이" 등 삽입
- 단, 과도한 감정 표현은 오히려 부자연스러우므로 1~2회 정도 적절히

## 3. 구조적 자연스러움 점검

### 문단 길이 변화
- 모든 문단이 3문장이면 AI 느낌이 강함
- 2문장 / 4문장 / 3문장 / 2문장처럼 변화를 줌

### 도입부 다양화
- 사실 진술: "2023년 여름, 인턴으로 합류한 첫 주에..."
- 질문형: "어떻게 하면 고객 이탈률을 줄일 수 있을까?"
- 대비: "숫자는 냉정했지만, 거기서 기회를 봤습니다."
- 에피소드: "새벽 2시, 서버가 터졌다는 알림이 울렸습니다."

### 마무리 다양화
- 교훈형: "이 경험은 ~를 가르쳐주었습니다" (최대 1회만)
- 연결형: "이 역량을 ~직무에서 발휘하겠습니다"
- 다짐형: "~을/를 실현하는 데 기여하겠습니다"
- 열린 결말: "그래서 지금도 매일 ~하고 있습니다"

## 4. 자연스러운 한국어 표현 팁

### ~하였습니다 vs ~했습니다
- "했습니다"가 더 자연스럽고 현대적
- "하였습니다"는 지나치게 격식적이라 어색할 수 있음
- 단, 한 편 내에서 통일

### 한자어 vs 고유어
- "실행하였다" → "해냈다", "해보았다"
- "수행하였다" → "맡았다", "진행했다"  
- 적절히 섞되, 고유어가 더 생동감 있음

### 수동태 최소화
- "~되었습니다" 반복 → "~했습니다"로 능동적 전환
- "선발되었습니다" → "선발 기회를 얻었습니다"
- "배치되었습니다" → "~팀에 합류했습니다"

## 5. 최종 AI 탐지 자가 점검

작성 완료 후 다음 질문에 스스로 답해본다:

1. 이 글에서 "저"를 빼고 다른 이름을 넣어도 어색하지 않은가? → YES면 차별화 부족
2. 모든 문장의 길이가 비슷한가? → YES면 AI 패턴
3. 접속부사("또한", "나아가", "더불어")가 3개 이상인가? → YES면 수정 필요
4. 구체적 숫자가 하나도 없는 문단이 있는가? → YES면 구체성 보강
5. 읽었을 때 특정 에피소드가 머릿속에 그려지는가? → NO면 생생함 부족
6. "열정", "노력", "최선"이 등장하는가? → YES면 행동으로 대체
