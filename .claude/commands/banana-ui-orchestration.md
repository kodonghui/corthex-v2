### Banana UI Orchestration Workflow

1. **Design Research**:
    - Query the user for the specific component/page to be designed.
    - Research current high-end dashboard trends (glassmorphism, neon dark mode, etc.) for `CORTHEX-v2`.

2. **Mockup Generation**:
    - Use the `generate_image` tool (powered by **Banana 2 / Gemini 3.1 Flash Image**) to create a 4K, hyper-realistic UI mockup.

3. **Design-to-Code Implementation**:
    - Use **Gemini 3.1 Pro High** to analyze the generated mockup's structure, colors, and components.
    - Write clean, responsive React/Next.js and Vanilla CSS code to implement the design exactly.

4. **Iterative Refinement**:
    - Request user feedback on the implementation and mockup.
    - Use Banana 2 for further image tweaks if necessary.
