# 한국 특화 AI 도구

> 조사일: 2026-03-11

---

## 1. 뤼튼 (Wrtn) — 한국어 AI 1위

- **사이트**: https://wrtn.ai
- **핵심 기능**:
  - **한국어 특화 생성형 AI** (GPT-4 + 네이버 HyperCLOVA 기반)
  - 콘텐츠 유형별 생성:
    - 블로그 포스트 (네이버 블로그 최적화)
    - SNS 캡션 (인스타, 페북, X)
    - 광고 카피
    - 이메일 마케팅
    - 보도자료
  - **뤼튼 도큐먼트**: 사업계획서/보고서 초안 작성
  - **뤼튼 Search**: 실시간 검색 기반 시장조사
  - 번역/요약/문체 변환
  - 한국 문화/관용어 반영 (자연스러운 한국어)
- **가격**: 기본 무료 (일부 고급 기능 유료)
- **추천 이유**: 한국어 품질이 ChatGPT보다 자연스러운 경우 많음. 무료라서 진입 장벽 제로

## 2. 미리캔버스 (MiriCanvas) — 한국형 Canva

- **사이트**: https://www.miricanvas.com
- **핵심 기능**:
  - 한국어 카드뉴스 템플릿 수천 개
  - AI 프레젠테이션 자동 생성
  - AI 이미지 생성
  - AI 라이팅
  - 인포그래픽 (플로차트, 차트, 타임라인)
  - 한국 감성 디자인 (한국 기업/교육기관 표준)
  - 한국어 폰트 라이브러리 풍부
- **가격**:
  - Free: 기본 템플릿+기능
  - Pro: 14,900원/월
  - Enterprise: 33,000원/월
- **추천 이유**: 한국에서 가장 많이 쓰는 디자인 도구. Canva보다 한국어 템플릿 압도적

---

## 3. 독스헌트 (DocsHunt) — 한국형 사업계획서 AI

- **사이트**: https://docshunt.ai
- **핵심 기능**:
  - 아이디어 입력 → 3분 만에 사업계획서 자동 생성
  - **한국 정부 지원사업 양식** 지원
  - 시장 분석, 페르소나, BM, 마케팅 전략 포함
  - 자료/이미지 첨부 반영
- **실적**: 누적 10,800건+ 생성
- **추천 이유**: 한국 정부 지원사업(창업 지원금) 신청용 계획서에 특화

---

## 4. 올거나이즈 Alli — 한국 기업 업무 자동화

- **사이트**: https://www.allganize.ai/ko
- **핵심 기능**:
  - 드래그앤드롭으로 기업용 AI 에이전트 제작
  - 직원/고객 문의 **75%+ 자동화** 달성
  - 문서 기반 Q&A 에이전트 (사내 규정, 매뉴얼 등)
  - 다국어 지원 (한/영/일)
  - 일부 카테고리 100% 자동화 사례 보유
- **가격**: Enterprise (개별 문의)
- **추천 이유**: 한국 대기업/중견기업 사내 업무 자동화 대표 솔루션. 실제 운영 사례 다수

## 5. 망고보드 (Mangoboard) — 한국 감성 디자인

- **사이트**: https://www.mangoboard.net
- **핵심 기능**:
  - AI 카드뉴스 이미지 생성
  - 웹툰/귀여운 캐릭터 스타일 (한국 특유 감성)
  - 인포그래픽 템플릿
  - 한국어 최적화
- **가격**: 무료~
- **추천 이유**: 귀여운/캐주얼 톤의 SNS 콘텐츠

## 6. 메이크봇 — LLM 통합 플랫폼

- **핵심 기능**:
  - 음성/채팅/챗봇/검색/내부업무 아우르는 LLM 통합
  - 한국 내 유일 실운영 수준 통합 LLM 플랫폼
  - 고객센터 챗봇, 내부 FAQ, 음성 안내 등
- **가격**: Enterprise (개별 문의)
- **추천 이유**: 콜센터/고객센터 운영하는 한국 기업용

## 7. Tyle.io — 카드뉴스+동영상 동시 제작

- **사이트**: https://tyle.io
- **핵심 기능**:
  - 카드뉴스와 동영상을 동시에 제작
  - 마케팅 특화
  - 한국어 지원
- **가격**: 별도 문의
- **추천 이유**: "같은 내용을 카드뉴스로도, 영상으로도" 필요한 경우

- **핵심 기능**:

  - 상업적 사용 무료
- **추천 이유**: 출판/독서/서평 콘텐츠 마케팅

## 한국 시장 AI 트렌드 (2026)

1. **"AI를 어디에 쓸 것인가"에 초점**: 글로벌은 "더 똑똑한 AI 만들기", 한국은 "AI 활용처 발굴"
2. **지능형 워크스페이스 보편화**: AI가 비서 역할. 2026년까지 글로벌 2000대 기업의 ~40%가 AI 에이전트와 직접 협업
3. **글로벌+국산 병행 사용**: ChatGPT/Notion AI/Midjourney 등 글로벌 도구 + 뤼튼/올거나이즈/미리캔버스 등 국산 도구
4. **CLOVA X (네이버) 서비스 종료 예정** (2026.4): 한국형 대화 AI 시장에서 뤼튼이 대안으로 부상
