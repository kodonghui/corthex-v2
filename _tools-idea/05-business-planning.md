## 사업기획서 자동 작성

### 1. 독스헌트 (DocsHunt) — 한국형 사업계획서 1위

- **사이트**: https://docshunt.ai
- **핵심 기능**:
  - 아이디어 입력 → **약 3분 만에** 완성도 높은 사업계획서 자동 생성
  - 포함 항목: 시장 분석, 페르소나, 비즈니스 모델, 마케팅 전략
  - **한국 정부 지원사업 양식** 지원 (창업 지원금 신청용)
  - 자료/이미지 첨부 반영 가능
  - 생성 후 섹션별 수정/보완 가능
- **실적**: 누적 10,800건+ 사업계획서 생성
- **대상**: 창업자, 소상공인, 정부 지원사업 신청자
- **추천 이유**: 한국 정부 지원사업 포맷에 맞춘 유일한 AI 도구

---

### 2. PrometAI — 재무모델 포함 완성형

- **사이트**: https://prometai.app
- **핵심 기능**:
  - **재무 예측(Financial Forecasting)**: 매출 예측, 손익계산서, 현금흐름표 자동 생성
  - **기업 가치 평가(Valuation)**: DCF, 멀티플 등 자동 계산
  - 산업별 벤치마크 데이터 제공 (경쟁사 대비 우리 위치)
  - 구조화된 사업계획서 생성
  - 투자자향 보고서 포맷
- **가격**: $55/월 (기본) / $145/월 (Pro)
- **추천 이유**: 재무 모델링까지 자동화. 투자 유치 준비 시 필수

---

### 3. Upmetrics — 사업계획서 + 피치덱 동시

- **사이트**: https://upmetrics.co
- **핵심 기능**:
  - 전통적 사업계획서 템플릿 + AI 보조
  - 동일 입력으로 **사업계획서와 피치덱 동시 생성**
  - 팀 협업 기능 (공동 편집)
  - 재무 예측 시트 내장
- **가격**: $9/월 (개인) ~ $49/월 (팀)
- **추천 이유**: 가격 대비 기능 우수. 초보자에게 적합

---

### 4. VentureKit — 심층 질문 기반 맞춤형

- **핵심 기능**:
  - 비즈니스 모델, 타겟 시장, 경쟁 환경에 대한 심층 질문
  - 답변 기반으로 맞춤형 사업계획서 생성
  - 가격 전략, 고객획득비용(CAC), 성장 가정 기반 재무 예측
- **추천 이유**: "질문에 답만 하면" 투자자급 사업계획서 완성

---

### 5. 15MinutePlan.ai — 15분 초안

- **핵심 기능**: 이름 그대로 15분 내 초안 완성
- **가격**: $29 (기본) ~ $79 (프리미엄, 다중 수정)
- **추천 이유**: 빠른 첫 초안이 필요할 때

## 피치덱 (투자 제안서 / IR 자료)

### 6. Gamma — 프레젠테이션 AI 1위

- **사이트**: https://gamma.app
- **핵심 기능**:
  - **AI Agent가 전체 과정 수행**: 웹 리서치 → 콘텐츠 정제 → 디자인 → 리스타일링
  - 자연어 대화로 수정 ("3번 슬라이드 톤을 더 전문적으로")
  - **Generate API** (2026.1 GA): 대규모 프로그래매틱 덱 생성 (API로 자동화 가능)
  - Adaptive Theme Logos: 배경색에 따라 로고 자동 전환 (밝은/어두운)
  - 파일 업로드/메모 붙여넣기/프롬프트 → 완성형 피치덱
  - 임베디드 공유 (웹사이트에 슬라이드 삽입)
  - 분석 대시보드 (누가 몇 번째 슬라이드까지 봤는지)
- **가격**:
  - Free: 400 AI 크레딧, Gamma 브랜딩 포함
  - Plus: $8/월
  - Pro: $15/월
  - Ultra: 최상위 (가격 미공개)
- **추천 이유**: 속도+품질 모두 최고. 프롬프트 → 5분 내 완성 피치덱

---

### 7. Beautiful.ai — 자동 레이아웃

- **사이트**: https://www.beautiful.ai
- **핵심 기능**:
  - **Smart Slide**: 콘텐츠 입력하면 레이아웃 규칙이 자동 적용
  - 브랜드 일관성 자동 유지 (로고, 색상, 폰트 자동 적용)
  - AI 기반 콘텐츠 생성
  - 디자인 감각 없어도 전문가 수준 슬라이드
- **가격**: Pro $12/월 (연간 결제)
- **추천 이유**: "디자인 감각이 없는데 깔끔한 슬라이드 필요" 한 경우

---

### 8. Slidebean — 실제 VC 구조 피치덱

- **사이트**: https://slidebean.com
- **핵심 기능**:
  - 120개 이상 **투자자 친화적** 피치덱 템플릿
  - 실제 VC가 원하는 슬라이드 구조에 맞춤
  - 전문 디자인 서비스 병행 (AI + 인간 디자이너)
- **추천 이유**: 실제 투자 유치 경험이 반영된 템플릿

---

### 9. Alai — 1000개 피치덱으로 학습된 AI

- **사이트**: https://getalai.com
- **핵심 기능**:
  - 1,000개 이상 실제 피치덱으로 학습된 AI
  - TAM 피라미드, 경쟁 매트릭스, 트랙션 타임라인 등 레이아웃 자동 이해
  - Nano Banana Pro 통합 → 고품질 이미지 기반 슬라이드
- **통계**: AI 생성 피치덱 사용 스타트업의 88%가 투자자 참여도 증가
- **추천 이유**: 투자자가 기대하는 구조를 AI가 알아서 맞춤

## 보고서 / 문서 자동화

### 10. Notion AI — Custom Agents로 반복 보고서 자동화

- **사이트**: https://www.notion.com
- **핵심 기능**:
  - **Custom Agents (2026 신기능)**: 반복 업무 자동화하는 자율 AI 팀원
    - 예: "매주 월요일 9시에 주간보고 초안 작성"
    - 예: "새 이슈 등록되면 자동으로 분류 + 담당자 배정"
    - 트리거(조건) 또는 스케줄(시간) 설정 → 24/7 자동 실행
  - AI Meeting Notes: 회의록 자동 전사 + 요약 + 액션아이템 추출
  - Research Mode: 주제 입력 → 상세 리포트 자동 생성
  - AI Blocks: 글쓰기 개선, 번역, 요약
  - MCP 통합: Linear, Figma, HubSpot 등 외부 도구 연결
- **가격**:
  - Business: $20/사용자/월 (연간)
  - Custom Agents: 2026.5.3까지 무료, 이후 1,000 크레딧당 $10
- **추천 이유**: 이미 Notion 쓰는 팀이라면 추가 도구 없이 AI 자동화

## 추천 워크플로우: 사업기획서 → 피치덱 → 투자 유치

```
1단계: 시장조사
  → Perplexity Deep Research ($20/월)로 시장 규모, 경쟁사, 트렌드 조사
  → 3분 만에 출처 50개 포함 보고서 완성

2단계: 사업계획서 작성
  → 독스헌트 (한국 정부지원사업용) 또는 PrometAI (투자자용)
  → 재무 예측 포함 완성형 계획서

3단계: 피치덱 생성
  → Gamma에 사업계획서 업로드 → 5분 내 피치덱 초안
  → 자연어로 수정 ("경쟁 분석 슬라이드를 표 형태로")

4단계: 정제
  → Claude/ChatGPT로 각 슬라이드 스크립트 정제
  → 발표 연습용 스크립트 생성

예상 총 비용: 약 $35~70/월
예상 총 시간: 기존 2주 → 반나절
```
