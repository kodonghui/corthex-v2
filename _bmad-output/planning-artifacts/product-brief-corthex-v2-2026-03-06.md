# Product Brief: corthex-v2

## Executive Summary

CORTHEX v2는 **AI 에이전트 29명으로 구성된 가상 기업 조직**을 CEO(사용자)가 지휘하는 올인원 AI 오케스트레이션 플랫폼이다. v1에서 8일 만에 실제 동작하는 29명 AI 조직을 완성했으나, 단일 사용자 한정이었다. v2는 이 검증된 핵심 기능을 **멀티테넌시 SaaS**로 확장하여, 여러 회사/팀이 각자의 AI 조직을 독립적으로 운영할 수 있게 한다.

핵심은 "CRUD 페이지"가 아니라 **실제로 일하는 AI 조직**이다. CEO가 "삼성전자 분석해줘"라고 말하면, 비서실장이 자동 분류하고, 투자분석팀장에게 위임하고, 전문가들이 도구를 사용해 병렬 작업한 뒤, 보고서를 종합해서 CEO에게 올린다. 이 전체 과정이 자동으로 돌아간다.

## Core Vision

### Problem Statement

기업과 개인이 AI를 업무에 활용하려 하지만, 현실은 **채팅창에서 1:1 대화**가 전부다. 복잡한 업무(투자 분석, 콘텐츠 제작, 법무 검토)를 처리하려면 여러 AI에게 각각 지시하고, 결과를 수동으로 종합해야 한다. AI가 도구를 쓰지 못하고, 서로 협업하지 못하고, 학습하지 못한다.

### Problem Impact

- **시간 낭비**: 동일한 지시를 여러 AI에게 반복. 결과 종합도 수작업
- **품질 불균일**: AI 응답의 품질을 체계적으로 관리할 방법 없음
- **도구 단절**: AI가 실시간 데이터(주가, 뉴스, API)에 접근 못함
- **지식 소실**: 이전 작업 결과가 다음 작업에 반영되지 않음
- **비용 블랙박스**: 어떤 작업에 얼마나 AI 비용이 드는지 파악 불가

### Why Existing Solutions Fall Short

| 기존 솔루션 | 한계 |
|------------|------|
| ChatGPT/Claude 채팅 | 1:1 대화만, 도구 호출 제한, 에이전트 간 협업 불가 |
| AutoGPT/CrewAI | 개발자 전용, UI 없음, 실제 업무 적용 어려움 |
| Microsoft Copilot | 특정 MS 생태계에 종속, 커스텀 에이전트 불가 |
| Zapier/Make | 워크플로우 자동화지만 AI 추론/판단이 없음 |

**핵심 갭**: "비개발자가 AI 조직을 구축하고 실제 업무를 시킬 수 있는 플랫폼"이 없다.

### Proposed Solution

CORTHEX v2는 **AI 에이전트를 기업 조직도처럼 구성**하여 CEO가 지휘하는 시스템이다:

1. **사령관실**: 자연어/슬래시/멘션으로 명령 → 비서실장이 자동 라우팅
2. **3계급 에이전트**: Manager(지휘) → Specialist(실행) → Worker(보조) 실제 위임 체인 + 자율 딥워크 (5단계 자율 작업: 계획→수집→분석→초안→보고서)
3. **Soul 시스템**: 에이전트별 성격/전문분야/판단원칙을 마크다운으로 정의, CEO가 웹 UI에서 직접 편집
4. **125개+ 도구**: 금융 API, 웹 검색, 차트 생성, SNS 발행 등 실제 호출
4. **품질 게이트**: 비서실장이 편집장 역할, 자동 검수 + 반려
5. **멀티 LLM**: Claude/GPT/Gemini 최적 모델 자동 배정 + Batch API 50% 할인
6. **AGORA 토론**: 6명 팀장이 라운드 기반 토론, SSE 실시간 스트리밍
7. **전략실**: 실시간 포트폴리오 + KIS 증권 자동매매
8. **스케치바이브**: 캔버스에서 다이어그램 + AI 실시간 편집
9. **멀티테넌시**: 회사별 완전 격리, 독립 에이전트 조직

### Key Differentiators

| 차별점 | 설명 |
|--------|------|
| **실제 동작하는 AI 조직** | 29명 에이전트가 실제로 위임/협업/보고. CRUD 목록이 아님 |
| **비개발자 친화** | 코드 없이 자연어로 AI 조직 지휘 |
| **도구 생태계** | 125개+ 실제 호출 가능한 외부 도구 (금융, 법무, SNS 등) |
| **품질 보장** | 자동 QA 게이트 + 재작업 루프. AI 환각 방지 |
| **비용 투명성** | 에이전트별/모델별/부서별 실시간 비용 추적 |
| **v1 검증 완료** | 8일 만에 29명 조직 완성, 실제 동작 검증된 아키텍처 |
| **AGORA 집단지성** | 멀티 에이전트 토론으로 단일 AI 한계 돌파 |
| **자율 딥워크** | 에이전트가 1회 답변이 아닌 5단계 자율 작업 (계획→수집→분석→초안→보고서) |
| **Soul 시스템** | 에이전트 성격/원칙을 CEO가 직접 편집. 부서별 표준 템플릿 제공 |

## Target Users

### Primary Users

#### 1. CEO/대표 — "AI 조직을 직접 지휘하는 사람"

**페르소나: 김대표 (42세, 중소기업 대표)**
- **배경**: 직원 15명 회사 운영. 투자, 마케팅, 법무 등 다방면 의사결정 필요
- **현재 고충**: ChatGPT를 매일 쓰지만 매번 같은 맥락 설명, 결과물 품질 들쭉날쭉
- **CORTHEX 사용법**: 사령관실에서 "이번 달 마케팅 전략 짜줘" → 비서실장이 CMO팀에 자동 위임 → CMO가 시장조사/콘텐츠/SEO 전문가에게 병렬 배분 → 종합 보고서 수신
- **핵심 니즈**: 한 번 지시하면 알아서 돌아가는 AI 조직
- **성공 기준**: "직원 채용 없이 29명 AI팀이 24시간 일한다"

#### 2. 개인 투자자 — "AI로 투자 분석을 자동화하는 사람"

**페르소나: 이투자 (35세, 직장인 겸 투자자)**
- **배경**: 한국/미국 주식 투자. 퇴근 후 분석 시간 부족
- **현재 고충**: 종목 분석을 AI에게 시키면 실시간 데이터 없이 일반론만 답변
- **CORTHEX 사용법**: "삼성전자 심층 분석" → CIO팀이 실시간 주가/공시/뉴스 수집 → 기술적 분석 + 리스크 평가 → 투자 의견 + 자동매매 실행
- **핵심 니즈**: 실시간 데이터 기반 AI 투자 분석 + 자동 실행
- **성공 기준**: "잠자는 동안에도 AI가 시장을 감시하고 매매한다"

#### 3. 콘텐츠 크리에이터 — "AI로 멀티 플랫폼 콘텐츠를 발행하는 사람"

**페르소나: 박크리 (28세, 1인 미디어)**
- **배경**: 인스타그램, 유튜브, 블로그를 운영하지만 콘텐츠 제작에 시간 부족
- **현재 고충**: 각 플랫폼별로 따로 AI에게 콘텐츠 작성 요청, 톤앤매너 불일치
- **CORTHEX 사용법**: "AI 트렌드 콘텐츠 5개 플랫폼에 발행" → CMO팀이 콘텐츠 생성 → 플랫폼별 최적화 → CEO 승인 → Selenium 자동 발행
- **핵심 니즈**: 한 번 지시로 5개 플랫폼에 일괄 발행
- **성공 기준**: "콘텐츠 제작 시간이 90% 줄었다"

### Secondary Users

#### 1. 관리자 (Admin)
- 회사 설정, 직원 관리, 에이전트 조직 구성
- Admin 전용 콘솔에서 멀티테넌시 관리
- 에이전트 성격(Soul) 편집, 도구 권한 설정

#### 2. 팀원 (직원)
- CEO가 초대한 같은 회사 구성원
- 자신만의 사령관실에서 AI에게 명령
- 사내 메신저로 동료와 소통
- 공유 지식 베이스 활용

### User Journey

```
1. 발견: "AI가 팀처럼 일한다고?" → 웹사이트/입소문
2. 가입: 회사 생성 + 기본 에이전트 29명 자동 배치
3. 첫 명령: 사령관실에서 "오늘 뉴스 브리핑 해줘" → 비서실장 자동 처리
4. Aha! 순간: 비서실장이 알아서 부서 배분 → 종합 보고서 → "이거 진짜 조직이다!"
5. 커스터마이징: 에이전트 성격 편집, 프리셋 등록, 도구 연결
6. 일상화: 크론 스케줄러로 매일 9시 자동 브리핑, 자동매매, SNS 자동 발행
7. 확장: 팀원 초대 → 각자 독립 워크스페이스 → 공유 지식 베이스
```

## MVP Scope

### Core Features (v1 기능 100% 이식 + 멀티테넌시)

MVP = v1에서 실제 동작했던 모든 기능을 v2 아키텍처로 이식. "v1에서 되던 건 v2에서도 반드시 된다."

#### Phase 1: 기반 인프라 (v2 신규)
- **멀티테넌시**: companyId 기반 완전 격리, JWT 인증
- **Admin 콘솔**: 회사/직원/에이전트 CRUD
- **실시간 통신**: WebSocket 7채널 멀티플렉싱, EventBus
- **보안**: AES-256-GCM 크리덴셜 볼트, 테넌트 격리

#### Phase 2: 핵심 엔진 (v1 이식 — 최우선)
- **오케스트레이터**: CEO 명령 → 비서실장 자동 분류 → 부서 위임 → 병렬 처리 → 종합 보고
- **에이전트 3계급**: Manager/Specialist/Worker 실제 동작 (CRUD 아님) + 자율 딥워크 5단계
- **Soul 시스템**: 에이전트 성격 마크다운 편집 + 부서별 표준 템플릿 (CEO 아이디어 #011)
- **처장 = 5번째 분석가**: Manager도 독자 분석 후 부하 결과와 종합 (CEO 아이디어 #007)
- **LLM 라우터**: Claude/GPT/Gemini 멀티 프로바이더 + 모델 자동 배정
- **도구 시스템**: ToolPool 125개+ 실제 호출, 에이전트별 권한 제어
- **사령관실**: 자연어 입력, @멘션, 슬래시 명령어 8종, 프리셋
- **품질 게이트**: 비서실장 편집장 역할, 자동 검수 5항목, 반려/재작업
- **비용 추적**: 에이전트별/모델별/부서별 실시간 비용 추적

#### Phase 3: 고급 기능 (v1 이식)
- **AGORA 토론**: 6명 팀장 라운드 기반 토론 (2/3라운드), SSE 스트리밍
- **전략실**: 포트폴리오 대시보드, 관심종목, KIS 자동매매, 모의거래
- **SNS 통신국**: 5개 플랫폼 발행, 승인 플로우, Selenium 자동화
- **스케치바이브**: Cytoscape 캔버스, Mermaid 양방향 변환, MCP SSE
- **에이전트 메모리**: 자동 학습 + 다음 작업 자동 참고
- **Batch API**: 50% 할인 비긴급 작업 일괄 처리

#### Phase 4: 운영 기능 (v1 이식)
- **작전현황 대시보드**: 요약 카드 4개, AI 사용량, 예산 관리, 퀵 액션
- **통신로그**: 활동/통신/QA/도구 4탭
- **작전일지**: 히스토리 검색, 북마크, A/B 비교, 리플레이
- **기밀문서**: 보고서 저장소, 등급별 필터
- **전력분석**: Soul Gym, 품질 대시보드, 에이전트 성능
- **크론기지**: Cron 스케줄러, 프리셋, 활성/비활성 토글
- **정보국**: RAG 문서 저장소, 드래그 업로드, 부서별 자동 주입
- **ARGOS**: 실시간 정보 수집, 트리거 기반
- **텔레그램 지휘**: 모바일 명령, @멘션, 부서별 보고

### Out of Scope for MVP

| 제외 기능 | 이유 | 향후 계획 |
|----------|------|----------|
| **랜딩 페이지/마케팅 사이트** | 제품 완성 후 별도 제작 | v2.1 |
| **결제/빌링 시스템** | 초기에는 무료 베타 | v2.2 |

### MVP Success Criteria

1. **기능 완전성**: v1 체크리스트 16개 항목 100% 동작 (stub/mock 아닌 실제 기능)
2. **오케스트레이션**: "삼성전자 분석" 명령 → 자동 위임 → 도구 호출 → 종합 보고서 (end-to-end)
3. **멀티테넌시**: 2개 회사가 독립적으로 각자 AI 조직 운영 가능
4. **성능**: 단순 명령 30초, 복합 명령 3분 이내 응답
5. **안정성**: 24시간 무중단 운영, 에러율 < 5%
