# Product Brief: CORTHEX v2

## Executive Summary

CORTHEX v2는 **AI 에이전트 조직을 직접 설계하고 운영하는 플랫폼**이다. CEO(사용자)가 자연어 한 줄로 명령하면, AI 에이전트 조직이 명령을 자동 분류하고, 적합한 부서에 위임하며, 전문가들이 병렬로 작업한 뒤, 결과를 종합하여 CEO 눈높이의 보고서를 반환한다.

v1은 29명 고정 조직으로 이 비전을 실증했다. 오케스트레이션, 125+ 도구 시스템, LLM 멀티 프로바이더 라우팅, AGORA 토론 엔진, KIS 자동매매, SketchVibe 캔버스, SNS 자동 발행, 품질 게이트, 작전현황 대시보드, 통신로그, 작전일지, 기밀문서, 전력분석, 자동화 워크플로우 등 22개 기능 영역이 실제 운영 환경에서 동작했다.

v2의 핵심 진화는 **동적 조직 관리**다. 29명 고정이 아니라, 관리자가 웹 UI에서 부서를 만들고, AI 직원과 Human 직원 워크스페이스를 자유롭게 생성-수정-삭제한다. 조직 구조 자체가 사용자의 비즈니스에 맞게 변형 가능한 것이 v2의 본질이다. 여기에 멀티테넌시(회사별 데이터 격리), 다중 사용자 인증, 사내 메신저를 추가하여 팀 단위 운영을 지원한다.

핵심 가치: **"나만의 AI 회사를 만들고, 운영하고, 진화시킨다."**

주요 타겟 사용자: 1인~중소기업의 대표/경영진/관리자로, AI를 팀처럼 운영하여 업무 생산성을 극대화하려는 비개발자 리더. (상세 페르소나는 Target Users 섹션에서 정의)

## Core Vision

### Problem Statement

현재 AI 도구 시장에는 근본적인 구조적 공백이 있다:

1. **1:1 대화형 AI** (ChatGPT, Claude): 한 번에 하나의 작업만 가능. "삼성전자를 재무적으로 분석하면서 동시에 기술적 분석도 하고, 리스크도 평가해줘"라고 하면 순차적으로만 처리. 부서 간 협업 개념 자체가 없다.

2. **자동화 플랫폼** (Zapier, Make): 정해진 트리거-액션 파이프라인만 실행. AI가 상황을 판단하고 자율적으로 도구를 선택하며 다단계 분석을 수행하는 것은 불가능.

3. **멀티에이전트 프레임워크** (AutoGPT, CrewAI, LangGraph): 실험적 수준. 프로덕션 UI 부재, 도구 통합 미흡, 조직 구조 설계 불가. 개발자만 사용 가능.

4. **v1 CORTHEX의 한계**: 29명 고정 조직은 유연성이 부족했다. 사용자마다 필요한 부서/에이전트 구성이 다른데, 조직 변경에 코드 수정이 필요했다. 단일 사용자 전용이라 팀/회사 단위 운영이 불가능했다.

**핵심 공백**: "여러 AI 에이전트가 조직 구조 안에서 자율적으로 협업하고, 비개발자가 그 조직을 자유롭게 설계하고 관리하는" 플랫폼이 시장에 존재하지 않는다.

### Problem Impact

이 공백이 만드는 실질적 피해:

- **업무 분절**: CEO가 투자 분석 결과를 마케팅 전략에 연결하려면 AI 도구 A에서 결과를 복사해서 AI 도구 B에 붙여넣어야 한다. 부서 간 정보 흐름이 수동이다.
- **반복 노동**: "매일 아침 시황 분석 + SNS 발행 + 일정 정리"를 매번 개별 명령으로 수동 요청. 자동화하려면 개발자가 코드를 짜야 한다.
- **품질 편차**: AI 응답을 그대로 사용하면 환각, 불완전한 분석, 형식 불일치가 발생. 품질 검수 체계가 없으면 CEO가 직접 검증해야 한다.
- **조직 경직**: 고정된 에이전트 구조는 비즈니스 변화에 대응하지 못한다. 새 프로젝트를 위해 "임시 TF팀"을 만들거나, 불필요한 부서를 해체하는 것이 불가능하다.
- **비용 불투명**: 어떤 에이전트가 얼마를 쓰는지, 어떤 모델이 가성비가 좋은지 추적할 수 없다. 예산 통제가 불가능.
- **팀 공유 불가**: 1인용 도구를 팀원과 공유할 수 없다. 같은 AI 조직을 여러 사람이 함께 쓰는 것이 구조적으로 지원되지 않는다.

### Why Existing Solutions Fall Short

| 기존 솔루션 | 할 수 있는 것 | 할 수 없는 것 |
|------------|-------------|-------------|
| ChatGPT / Claude | 1:1 대화, 단일 작업 처리 | 다부서 병렬 작업, 위임 체인, 도구 125+개 통합, 조직 구조 설계 |
| Zapier / Make | 트리거-액션 자동화 | AI 자율 판단, 다단계 분석, 실시간 대화형 명령 |
| AutoGPT / CrewAI | 멀티에이전트 실험 | 프로덕션 UI, 안정적 도구 통합, 비개발자 사용, 비용 관리 |
| Custom GPTs / Assistants | 단일 에이전트 커스텀 | 에이전트 간 위임, 계급 구조, 병렬 작업, 조직 관리 |
| Notion AI / Copilot | 문서/코드 내 AI 지원 | 독립적 에이전트 조직, 외부 도구 통합, 자율 작업 수행 |

**결론**: 기존 솔루션들은 "AI를 도구로 쓴다". CORTHEX는 "AI를 조직으로 운영한다". 이것은 근본적으로 다른 패러다임이다.

### Proposed Solution

CORTHEX v2는 CEO에게 **AI 기업**을 제공한다. 핵심 구성요소:

**1. 동적 조직 관리 (v2 핵심 신규 기능)**
- 관리자가 웹 UI에서 부서를 자유롭게 생성/수정/삭제
- AI 직원: 계급(Manager/Specialist/Worker), LLM 모델, 허용 도구, Soul(성격 문서)을 지정하여 생성
- Human 직원: 워크스페이스를 생성하여 같은 AI 조직에 접속. 비용/데이터 격리
- 조직도 실시간 시각화: 트리 뷰로 부서-팀장-전문가 계층 표시, 드래그로 부서 이동, 클릭으로 에이전트 상세 정보(Soul/도구/모델/비용) 확인, 실시간 작업 상태(유휴/작업중/에러) 색상 표시
- 조직 템플릿: "투자분석 조직", "마케팅 조직" 등 프리셋으로 빠른 시작
- **시스템 에이전트 보호**: 비서실장(Chief of Staff) 등 오케스트레이션 필수 역할은 "시스템 에이전트"로 지정되어 삭제 불가. 사용자가 자유롭게 CRUD할 수 있는 "사용자 에이전트"와 구분
- **조직 변경 cascade 처리**: 부서 삭제 시 해당 부서의 진행 중 작업은 자동 완료 대기 또는 CEO 확인 후 강제 종료, 에이전트 메모리/학습 기록은 아카이브 보존, 비용 기록은 영구 보존(회계 추적), 하위 에이전트는 미배속 상태로 전환(재배치 가능)

**2. 사령관실 (Command Center) -- v1 검증 완료**
- 일반 텍스트: "삼성전자 분석해줘" -> 비서실장이 자동 라우팅
- @멘션: "@투자분석팀장 삼성전자 분석" -> 특정 팀장 직접 지정
- 슬래시 명령어 8종: /전체, /순차, /도구점검, /배치실행, /배치상태, /명령어, /토론, /심층토론
- 프리셋: 자주 쓰는 명령을 단축어로 저장
- 실시간 위임 체인 표시: "비서실장 -> CIO -> 종목분석가" 진행 과정을 WebSocket으로 스트리밍

**3. 비서실장 오케스트레이션 -- v1 검증 완료**
- 명령 자동 분류: AI가 명령을 분석하여 담당 부서 결정
- 팀장(Manager)에게 위임 -> 팀장이 전문가(Specialist)들에게 병렬 배분
- 처장 = 5번째 분석가: Manager도 독자적으로 분석한 뒤 부하 결과와 종합 (CEO 아이디어 #007)
- 비서실장 = 편집장: 최종 결과를 CEO 눈높이로 정리 + 5항목 품질 검수 (CEO 아이디어 #010)
- 실패 시 재작업 자동 지시

**4. 3계급 에이전트 시스템 -- v1 검증 완료**
- Manager (Sonnet): 업무 분해 + 배분 + 종합 + 독자 분석
- Specialist (Haiku): 실제 작업 수행 + 도구 사용 + 자율 딥워크 (계획 -> 데이터 수집 -> 분석 -> 초안 -> 최종)
- Worker (Haiku): 단순 반복 (요약, 일정 관리, 데이터 정리)
- Soul 시스템: 마크다운 기반 성격/전문분야/판단원칙 정의, 웹 UI에서 편집
- 부서별 표준 템플릿: AI도 구조가 있으면 더 잘 쓴다 -- 부서마다 보고서 형식/분석 프레임워크 표준화 (CEO 아이디어 #011)

**5. 도구 시스템 (125+개) -- v1 검증 완료**
- 금융(kr_stock, dart_api, sec_edgar, backtest_engine), 법무(law_search, contract_reviewer), 마케팅(sns_manager, seo_analyzer), 기술(uptime_monitor, security_scanner), 공통(real_web_search, spreadsheet_tool, chart_generator, email_sender)
- 에이전트별 allowed_tools 권한 제어
- 도구 호출 로그 전체 기록, UI에서 확인 가능
- 예측 워크플로우 테스트: 코드 전수검사 -> 예측 다이어그램 -> 선제 수정으로 도구 안정성 사전 확보 (CEO 아이디어 #004)

**6. LLM 멀티 프로바이더 라우터 -- v1 검증 완료**
- Anthropic (Claude Opus/Sonnet/Haiku), OpenAI (GPT-5/5-mini/5.2), Google (Gemini 3.1 Pro/2.5 Flash)
- 에이전트별 모델 배정 (계급에 따라 자동 추천)
- Batch API: 비긴급 작업 50% 할인, BatchCollector가 큐에 모아 일괄 실행

**7. AGORA 토론 엔진 -- v1 검증 완료**
- 팀장급 에이전트들이 주제에 대해 라운드 기반 토론 (2라운드 일반 / 3라운드 심층)
- SSE 실시간 스트리밍으로 토론 과정 표시
- Diff 뷰 + 책 형식 정리
- CEO는 관전만 -- 최종 합의/비합의 결과 수신

**8. 전략실 (Trading) -- v1 검증 완료**
- 실제 포트폴리오 대시보드 (현금, 보유종목, 수익률, 총자산, 오늘/누적 수익)
- 관심종목: 드래그 정렬, 시장 필터(KR/US), 60초 자동 갱신, 클릭 시 차트 모달
- KIS 자동매매: CIO(분석) + VECTOR(실행봇) 분리 구조 (CEO 아이디어 #001). CIO가 신뢰도 -> 비중% 자율 산출
- 투자 성향 선택: 보수/균형/공격
- 모의거래 (Paper Trading): 초기자금 설정 + 실제 시장 데이터로 가상 매매

**9. SketchVibe 캔버스 -- v1 검증 완료 (CEO 아이디어 #009)**
- Cytoscape.js 기반 8종 노드 캔버스 (agent, system, api, decide, db, start, end, note)
- 드래그 이동, 더블클릭 이름 편집, Delete 삭제, Space바 연결 모드, Ctrl+클릭 멀티선택
- Mermaid <-> Cytoscape 양방향 변환 (classDef/style 커스텀 색상, subgraph -> compound parent)
- MCP SSE 연동: AI가 실시간으로 캔버스에 노드 추가/수정, read_canvas()로 AI가 캔버스 상태 읽기
- 저장/불러오기: 확인된 다이어그램 저장 + 지식 베이스에서 flowchart 불러오기

**10. 작전현황 (홈 대시보드) -- v1 검증 완료**
- 요약 카드 4개: 오늘 작업(총/완료/진행), 오늘 비용($X.XXXX + 누적), 에이전트 수(N명), 외부 연동 상태(Notion 등)
- AI 사용량: 프로바이더별 호출 횟수 세로 막대 그래프 (Anthropic/OpenAI/Google)
- 예산 관리: 일일 한도 / 월 한도 진행 바 (초록->노랑->빨강)
- 퀵 액션: 루틴 명령(전 부서 현황, 오늘 일정, 예산 현황) + 시스템 명령(도구 점검, 대기 전송, 배치 상태) + 최근 사용 명령어
- CEO 만족도: 썸업/다운 피드백 + 만족률 원형 차트

**11. 통신로그 (Activity Log) -- v1 검증 완료**
- 4개 서브탭으로 모든 내부 활동 추적:
  - 활동 탭: 에이전트별 활동 기록 (시작/완료/에러)
  - 통신 탭: 내부 위임 기록 (from -> to, 메시지 내용, 비용, 토큰 수)
  - QA 탭: 품질 검수 결과 (통과/반려/재작업 이력)
  - 도구 탭: 도구 호출 기록 (어떤 에이전트가 어떤 도구를 호출했는지, 결과, 소요 시간)

**12. 작전일지 (History) -- v1 검증 완료**
- 모든 CEO 명령 + 결과 히스토리 영구 보존
- 검색/필터: 상태별, 날짜별, 텍스트 검색
- 북마크, 태그, 아카이브 기능
- A/B 비교: 두 작업 결과를 나란히 비교
- 리플레이: 동일 명령을 동일 조건으로 재실행

**13. 기밀문서 (Archive) -- v1 검증 완료**
- 보고서 저장소: 에이전트가 생성한 모든 보고서를 체계적으로 보관
- 부서별, 등급별 필터 (confidential, internal, public 등)
- 유사 문서 찾기 (correlation): 관련 보고서를 자동 연결

**14. 전력분석 (Performance) -- v1 검증 완료**
- Soul Gym (AI 진화): 에이전트별 개선 제안, 신뢰도 점수, 필요 토큰 추정
- 품질 대시보드: 총 리뷰 수, 통과율, 평균 점수, 실패 작업 목록 + 사유
- 에이전트별 성능: 호출 수, 성공률, 평균 비용, 평균 응답 시간

**15. 자동화 (Workflow) -- v1 검증 완료**
- 워크플로우 CRUD: 다단계 자동화 파이프라인 정의
- 스텝 정의: 데이터 수집 -> 분석 -> 보고서 생성 등 순차/병렬 스텝 구성

### Key Differentiators

1. **동적 조직 설계**: 시장 유일. 다른 멀티에이전트 도구들은 에이전트를 코드로 정의한다. CORTHEX는 비개발자가 웹 UI에서 부서/직원/에이전트를 자유롭게 조합하여 "나만의 AI 회사"를 만든다. 부서 삭제 시 진행 중 작업/메모리/비용 기록의 cascade 처리까지 설계되어 있다. 조직 구조 자체가 제품의 핵심 가치이다.

2. **v1 실전 검증된 오케스트레이션**: 위임 체인(CEO -> 비서실장 -> 팀장 -> 전문가 -> 도구)이 실제 운영 환경에서 동작한 것이 검증됨. 125+ 도구, LLM 라우팅, 토론 엔진, 자동매매가 프로덕션 수준으로 작동. 실험적 프레임워크가 아니라 검증된 시스템이다.

3. **비개발자 완전 자율 운영**: 부서 생성, 에이전트 추가, Soul(성격) 편집, 도구 권한 변경, 조직 구조 재편, 부서별 표준 템플릿 설정이 전부 웹 UI에서 가능. 코드 한 줄 안 건드리고 AI 조직을 운영한다.

4. **3계급 비용 최적화**: Manager(Sonnet), Specialist(Haiku), Worker(Haiku)로 계급별 모델을 자동 배정. Batch API 50% 할인. 에이전트별/부서별 비용 추적으로 예산을 실시간 관리. 이것은 다른 멀티에이전트 도구에 없는 기업 운영 관점이다.

5. **품질 게이트 내장**: 비서실장이 편집장 역할로 모든 결과물을 5항목(결론/근거/리스크/형식/논리) 자동 검수. 불합격 시 자동 재작업. AI 환각과 품질 편차를 시스템적으로 통제한다.

6. **멀티테넌시 팀 운영**: 같은 AI 조직을 팀원들과 공유하되, 각 사용자의 비용/데이터는 완전 격리. 관리자가 팀원별 권한과 접근 범위를 통제. 사내 메신저로 Human 직원 간 협업까지 지원. 1인용이 아닌 기업용 AI 플랫폼이다.

- 사내 메신저로 다른 Human 직원과 소통하며, AI 분석 결과를 공유하거나 협업

**외부 연동 시스템**
- 텔레그램 봇: 모바일에서 CEO/직원이 AI 조직에 명령
- KIS 증권 API: 자동매매 실행 시 주문 전송/체결 확인
- Notion/Google 등: 외부 도구와 양방향 데이터 연동

### User Journey

**1. 발견 (Discovery)**
- "AI로 회사 운영 자동화", "AI 에이전트 조직", "멀티에이전트 플랫폼" 검색 -> CORTHEX 발견
- "ChatGPT 여러 명이 동시에 일하는 느낌" 이라는 핵심 메시지에 흥미
- 데모 영상에서 "삼성전자 분석" 명령 -> 4명 에이전트가 병렬 작업하는 과정을 실시간으로 보고 "이건 다르다" 인식

**2. 온보딩 (First 10 Minutes)**
- **CEO/개인 경로**: 회원가입 -> 조직 템플릿 선택 화면 제시 (투자분석 조직 / 마케팅 조직 / 올인원 / 빈 조직) -> "투자분석 조직" 선택 -> 비서실장 + CIO + 4명 전문가가 자동 생성 -> 즉시 사령관실 진입 -> 온보딩 가이드: "아래 명령을 입력해보세요: 삼성전자 분석해줘" -> 첫 명령 실행 -> 위임 체인 실시간 표시 -> 보고서 수신 -> "이게 이렇게 되는구나" 이해
- **Admin 경로**: 회원가입 -> 관리자 콘솔 진입 -> 회사 정보 설정 -> 조직 설계 시작 (부서 생성 -> AI 에이전트 배치 -> Soul/도구/모델 설정) -> Human 직원 워크스페이스 생성 + 접근 권한 부여 -> 비용 한도 설정 -> 팀원들에게 초대 링크 발송. 관리자는 "조직 만들기"가 첫 경험이고, 사령관실 명령은 그 다음이다

**3. 핵심 사용 (Daily Usage)**
- 매일 아침: 사령관실에서 "오늘 시황 브리핑" -> 크론으로 자동화하면 출근 전에 텔레그램으로 수신
- 업무 중: "@마케팅팀장 이번 주 SNS 콘텐츠 5개 작성해줘" -> 승인 플로우 -> 자동 발행
- 의사결정: "/토론 신사업 진출 A안 vs B안" -> 6명 팀장이 각자 관점에서 토론 -> 합의/비합의 결과
- 분석: "삼성전자 vs TSLA 비교 분석" -> 병렬 분석 -> A/B 비교 뷰 제공
- 전략실: 포트폴리오 대시보드 확인 -> 자동매매 승인/거부 -> 모의거래로 전략 테스트

**4. Aha! 순간 (Value Realization)**
- **첫 번째 Aha!**: "삼성전자 분석" -> 4명이 동시에 분석 -> 종합 보고서 -> "이건 진짜 팀이 일하는 것 같다!"
- **두 번째 Aha!**: 관리자 콘솔에서 "법무부" 생성 -> 에이전트 2명 배치 -> 즉시 "@법무팀장 계약서 검토해줘" 가능 -> "조직을 내가 만들 수 있다니!"
- **세 번째 Aha!**: 품질 게이트가 에이전트 보고서를 반려하고 자동 재작업 -> 개선된 결과 수신 -> "AI가 알아서 품질도 관리해주네"
- **네 번째 Aha!**: 비용 대시보드에서 부서별/에이전트별 비용을 보고 Haiku로 모델을 변경 -> 비용 40% 절감 -> "이게 기업 운영이지"

**5. 장기 사용 (Power User Evolution)**
- **조직 진화**: 비즈니스 변화에 맞춰 부서 신설/해체, 에이전트 추가/제거, Soul 편집으로 전문분야 조정
- **자동화 심화**: 프리셋 10+개 저장, 크론 스케줄러로 루틴 명령 5+개 자동화, 워크플로우로 다단계 파이프라인 구축
- **팀 확장**: Human 직원 워크스페이스 추가, 부서별 접근 권한 세분화, 사내 메신저로 AI 결과 기반 협업
- **데이터 축적**: 작전일지에 수백 건의 명령 이력 축적 -> 리플레이/A/B 비교로 의사결정 개선, 기밀문서에 보고서 아카이브
- **AI 진화**: Soul Gym에서 에이전트별 개선 제안 확인 -> Soul 업데이트 -> 에이전트 성능 점진적 향상, 에이전트 메모리가 축적되어 반복 작업의 품질 자동 상승

## MVP Scope

### Core Features (Phase 1 MVP -- 반드시 포함)

v1-feature-spec 22개 기능 중 핵심 오케스트레이션 + v2 핵심 차별점(동적 조직 관리)을 우선 구현. 이 기능들이 동작해야 "AI 조직이 실제로 일한다"는 핵심 가치를 전달할 수 있다.

**P0: 동적 조직 관리 (v2 핵심 신규)**
- 부서 CRUD: 웹 UI에서 부서 생성/수정/삭제
- AI 직원 CRUD: 계급(Manager/Specialist/Worker), LLM 모델, 허용 도구, Soul(성격 문서) 지정하여 생성/수정/삭제
- Human 직원 워크스페이스 CRUD: 직원 초대, 접근 권한 부여, 비용/데이터 격리
- 조직도 시각화: 트리 뷰 + 실시간 상태(유휴/작업중/에러) + 드래그 이동 + 클릭 상세
- 시스템 에이전트(비서실장 등) 보호: 삭제 불가, 사용자 에이전트와 구분
- 조직 변경 cascade 처리: 부서 삭제 시 진행 중 작업/메모리/비용 기록 안전 처리
- 조직 템플릿: "투자분석", "마케팅" 등 프리셋으로 빠른 시작

**P0: 사령관실 (Command Center) -- v1 기능 #1**

### MVP Success Criteria

MVP가 성공했다고 판단하는 기준 -- 아래 **6개 조건을 모두 충족**해야 Phase 2 진행:

**1. 핵심 오케스트레이션 동작 검증**
- CEO가 사령관실에서 자연어 명령 입력 -> 비서실장 자동 분류 -> 팀장 위임 -> 전문가 병렬 작업 -> 결과 종합 -> 보고서 반환이 **실제로 동작**
- 오케스트레이션 성공률 > 95%
- 평균 응답 시간: 단순 < 60초, 복합 < 5분

**2. 동적 조직 관리 동작 검증**
- 관리자가 웹 UI에서 부서/에이전트/Human 직원을 자유롭게 생성/수정/삭제 가능
- 조직 구조 변경 후 즉시 사령관실에서 새 조직으로 명령 가능
- 시스템 에이전트(비서실장) 보호 + cascade 처리 정상 동작

**3. 도구 시스템 실제 호출**
- 125+ 도구 중 핵심 30+ 도구가 에이전트에 의해 실제 호출되어 결과 반환
- 도구 호출 성공률 > 90%
- 에이전트별 도구 권한 제어 정상 동작

**4. 멀티테넌시 격리 검증**
- companyId 기반 데이터 완전 격리 확인 (A회사 데이터가 B회사에 노출되지 않음)
- JWT 인증 + 사용자별 워크스페이스 격리 동작
- 2+ 테스트 회사 동시 운영 성공

**5. 비용 통제 동작 검증**
- 3계급 모델 배정(Manager=Sonnet, Specialist=Haiku)이 실제 비용 40%+ 절감 확인
- 일일/월 예산 한도 설정 + 초과 경고 정상 동작
- 에이전트별/부서별 비용 대시보드 정확도 확인

**6. 사용자 가치 확인**
- 내부 사용자(CEO) 일일 10+ 명령 달성 (활발한 사용 의사)
- CEO 만족도 thumbs up 70%+ (첫 3개월 기준, Phase 2에서 80%+ 목표)
- "이건 ChatGPT와 다르다" -- 병렬 작업 + 위임 체인의 가치를 체감
