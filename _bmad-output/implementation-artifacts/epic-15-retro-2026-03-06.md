## 교훈

### 1. 통합 테스트 환경 부재
- 1152~1252개 유닛 테스트는 통과하지만, 144개 통합 테스트는 실행 서버 필요
- 실제 DB 연동 테스트 없이 API 라우트를 모킹으로만 검증
- Epic 16 이후 통합 테스트 환경 구축 고려 필요

### 2. CodeMirror 번들 사이즈
- 15-3에서 CodeMirror가 607KB (gzip 208KB) 별도 청크로 로드
- React.lazy()로 초기 로드 영향은 최소화했지만, 모바일 환경에서 체감될 수 있음
- 향후 번들 최적화 스프린트에서 tree-shaking 검토

### 3. 에러 카운터 인메모리 한계
- 15-4 에러 카운터가 서버 재시작 시 초기화됨
- 프로덕션에서는 영속적 에러 로깅이 필요하지만, 현재 P3 수준에서는 충분
- Epic 16+ 에서 외부 로깅 서비스 연동 가능

### 4. Admin UI 페이지 증가
- 소울 템플릿, 모니터링, 조직도 추가로 사이드바 메뉴 11개
- 그룹핑/카테고리화 없이 나열 중 — UX 개선 여지
