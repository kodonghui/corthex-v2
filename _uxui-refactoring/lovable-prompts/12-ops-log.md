# 12. 작전일지 (Operation Log) 페이지

> Route: `/ops-log` — CEO App

---

## What This Page Is For

CEO가 내린 모든 명령(Command)의 실행 기록을 검색, 필터링, 정렬하여 조회하는 페이지입니다. 각 명령이 어떤 에이전트에게 전달되었고, 어떤 결과를 냈고, 품질 점수는 얼마이며, 소요 시간은 얼마인지 한눈에 파악합니다. 중요한 명령은 북마크하여 빠르게 재접근할 수 있습니다. CSV 내보내기도 지원합니다.

---

## Data Displayed

### 명령 목록 (메인 테이블/리스트)
- **명령 텍스트** (text) — 사용자가 입력한 명령 원문
- **유형** (type) — direct, delegation, deep-work 등
- **상태** (status) — pending, processing, completed, failed, cancelled
- **대상 에이전트** (targetAgentName) — 명령이 전달된 에이전트 이름 (없을 수도 있음)
- **소속 부서** (targetDepartmentName) — 대상 에이전트의 부서
- **결과 미리보기** (result) — 최대 500자로 잘린 결과 텍스트
- **품질 점수** (qualityScore) — 0~10 스케일 실수. 5개 카테고리(결론 품질, 근거 출처, 위험 평가, 형식 준수, 논리 일관성) 평균
- **품질 결론** (qualityConclusion) — approved, needs-rework 등
- **소요 시간** (durationMs) — 밀리초 단위, 사람이 읽기 좋게 변환 필요
- **생성일** (createdAt), **완료일** (completedAt)
- **북마크 여부** (isBookmarked) + 북마크 메모 (bookmarkNote)

### 명령 상세 (개별 항목 선택 시)
- 위의 모든 필드 + **결과 전문** (잘리지 않은 result)
- **메타데이터** (metadata) — 슬래시 명령 종류, 프리셋 ID 등 추가 정보

### 페이지네이션
- **현재 페이지, 총 건수, 페이지당 항목 수** (page, total, limit)

---

## User Actions

### 검색/필터
1. **텍스트 검색** — 명령 텍스트, 결과 텍스트, 에이전트 이름에서 키워드 검색 (ILIKE)
2. **날짜 범위 필터** — 시작일~종료일 지정
3. **에이전트 필터** — 특정 에이전트의 명령만 보기
4. **부서 필터** — 특정 부서의 에이전트 명령만 보기
5. **유형 필터** — direct/delegation/deep-work 등
6. **상태 필터** — pending/processing/completed/failed/cancelled
7. **품질 점수 범위 필터** — 최소~최대 점수 지정
8. **북마크만 보기** — 내가 북마크한 명령만 필터

### 정렬
9. **정렬 기준 변경** — 날짜(기본), 품질 점수, 소요 시간 기준으로 오름차순/내림차순

### 북마크
10. **북마크 추가** — 특정 명령을 북마크 (메모 선택 입력). 이미 북마크된 것은 중복 방지
11. **북마크 메모 수정** — 기존 북마크의 메모 텍스트 변경
12. **북마크 삭제** — 북마크 해제
13. **북마크 목록 조회** — 북마크된 항목만 별도 목록으로 보기 (페이지네이션 지원)

### 내보내기
14. **CSV 내보내기** — 현재 필터 조건에 해당하는 명령 목록을 CSV로 다운로드 (최대 1000건)

### 상세 보기
15. **명령 상세 조회** — 목록에서 항목을 선택하면 전체 결과, 메타데이터, 품질 상세를 볼 수 있음

## UX Considerations

- Employee 역할 사용자는 본인 소속 부서(departmentIds)의 명령만 볼 수 있습니다. 이 제한은 서버에서 처리되지만, 필터 UI에서도 접근 불가 부서는 비활성화하세요.
- 품질 점수는 시각적 인디케이터(높음/중간/낮음)로 표현하면 좋습니다.
- 상태별로 시각적 구분(아이콘이나 레이블 스타일)이 필요합니다.
- 검색+필터 조합이 많으므로, 적용된 필터를 한눈에 보여주는 영역이 필요합니다.
- durationMs는 "3.2초", "1분 23초" 등 읽기 쉬운 형태로 변환하세요.
- 결과 미리보기(500자)에서 전문을 보려면 상세 보기로 진입해야 합니다.
- 북마크된 항목은 목록에서 즉시 식별 가능해야 합니다.
- CSV 내보내기는 현재 필터를 유지하면서 다운로드합니다.
- 빈 상태: 명령이 없을 때, 검색 결과가 없을 때, 북마크가 없을 때 각각 안내 필요.

---

## What NOT to Include

- 색상, 폰트, 크기, px/rem 등 시각적 스타일 지정
- 특정 컴포넌트 라이브러리나 UI 프레임워크 언급
- 레이아웃 배치 지정
- 오케스트레이션 태스크 세부 체인 시각화 (이 페이지에서는 요약만)
- 비용(totalCostMicro)은 현재 null로 반환되므로 표시하지 않음
