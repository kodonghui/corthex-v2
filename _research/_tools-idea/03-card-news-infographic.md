# 카드뉴스 / 인포그래픽 AI 생성기

## 한국 특화 도구

### 1. 미리캔버스 (MiriCanvas) — 한국어 카드뉴스 1위

- **사이트**: https://www.miricanvas.com
- **핵심 기능**:
  - **한국어 카드뉴스 템플릿 압도적** (수천 개)
  - AI 프레젠테이션 자동 생성
  - AI 이미지 생성 (프롬프트 → 이미지)
  - AI 라이팅 (텍스트 자동 작성)
  - 인포그래픽 전문 도구: 플로차트, 차트, 타임라인
  - 인스타그램/블로그/유튜브 썸네일 최적 사이즈 프리셋
  - 팀 협업 기능
- **가격**:
  - Free: 기본 템플릿+기능
  - Pro: 14,900원/월 (연간 결제 시 160,800원/년)
  - Enterprise: 33,000원/월
- **왜 1위인가**: 한국 기업/교육기관/공공기관의 사실상 표준 디자인 도구. 한국어 폰트, 한국 감성 템플릿이 압도적

---

### 2. Mirra — AI 카드뉴스 전문

- **사이트**: https://www.mirra.my
- **핵심 기능**:
  - AI 캐러셀(카드뉴스) 생성 특화
  - **5분 만에 전문가급 카드뉴스** 완성
  - 멀티플랫폼 최적화 (인스타, 링크드인, X 등)
  - 텍스트 입력 → 슬라이드 구성 + 디자인 자동
- **가격**: Free ~ $20/월
- **추천 이유**: 카드뉴스만 전문적으로 만들고 싶을 때

---

### 3. 망고보드 (Mangoboard)

- **사이트**: https://www.mangoboard.net
- **핵심 기능**:
  - AI 카드뉴스 이미지 생성
  - 한국어 최적화
  - **웹툰/귀여운 캐릭터 스타일** 특화 (한국 감성)
  - 인포그래픽 템플릿
- **가격**: 무료~
- **추천 이유**: 귀여운/캐주얼한 톤의 카드뉴스에 적합

### 4. 캐럿 AI (Carat)

- **핵심 기능**:
  - 책 소개만 입력하면 인스타 피드 카드뉴스/썸네일/포스터 자동 생성
  - **상업적 사용 무료**
  - 북스타그램, 서평 콘텐츠 특화

- **추천 이유**: 출판/서평/독서 콘텐츠 마케팅

## 글로벌 도구

### 5. Canva Magic Studio — 글로벌 1위

- **사이트**: https://www.canva.com
- **사용자**: 1.9억+ MAU (월간 활성 사용자)
- **핵심 기능**:
  - **25개 이상 AI 도구 통합**:
    - Magic Write: AI 텍스트 생성/편집
    - Magic Media: 텍스트 → 이미지/비디오 생성 (Google Veo 3 기반)
    - Magic Edit: 이미지 내 특정 부분만 AI로 변경
    - Magic Expand: 이미지 영역 확장
    - Magic Grab: 이미지 내 오브젝트 분리
    - AI 배경 제거 (원클릭)
    - Magic Resize: 1개 디자인 → 여러 사이즈 자동 변환
    - Magic Design: 텍스트 프롬프트 → 소셜 포스트 세트 자동 생성
  - 1억+ 프리미엄 스톡 이미지/영상
  - 인스타그램 직접 퍼블리싱 (캔바 내에서 바로 게시)
  - 한국어 인터페이스 지원
- **가격**:
  - Free: 월 ~50회 AI 사용
  - Pro: $12.99/유저/월 (월 ~500회 AI 사용)
  - Teams: $14.99/유저/월 → 2026년 $100/유저/년으로 인상 논란
- **추천 이유**: 카드뉴스+프레젠테이션+영상+소셜포스트 전부 하나에서 해결

---

### 6. Adobe Express — 포토샵 생태계 연동

- **사이트**: https://www.adobe.com/express
- **핵심 기능**:
  - 프롬프트로 Instagram 포스트/스토리/광고 템플릿 자동 생성
  - Adobe Stock 연동
  - Photoshop/Illustrator 파일 직접 편집
  - 브랜드 키트 (로고, 색상, 폰트 일관 적용)
- **가격**:
  - Free: 기본 기능
  - Premium: $9.99/월
- **추천 이유**: 이미 Adobe 생태계(포토샵, 일러스트레이터) 사용 중인 팀

---

### 7. Visme — 데이터 시각화 + 스토리텔링

- **사이트**: https://www.visme.co
- **핵심 기능**:
  - AI 인포그래픽 자동 생성
  - 프레젠테이션 생성
  - 데이터 시각화 특화 (차트, 그래프, 위젯 풍부)
  - 스토리텔링 기반 레이아웃
  - 인터랙티브 콘텐츠 (애니메이션, 팝업)
- **가격**:
  - Free: 3 프로젝트
  - Starter: $29/월
  - Pro: $59/월
- **추천 이유**: 데이터가 많은 보고서형 인포그래픽

- **사이트**: https://piktochart.com
- **핵심 기능**:

  - 데이터 시각화

  - Free: 기본 기능
  - Pro: $14/월~
