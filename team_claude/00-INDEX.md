## 핵심 요약 (3줄)

1. **팀 에이전트 = 독립 프로세스** — 서브에이전트(같은 프로세스)와 달리, 각 팀원이 별도 Claude Code 인스턴스로 실행됨
2. **P2P 통신 가능** — 팀원끼리 직접 메시지 주고받을 수 있어서 **토론/디베이트** 가능 (서브에이전트는 불가)
3. **tmux 모드에서 팀장 죽어도 팀원 살아남음** — 하지만 좀비 프로세스가 되므로 수동 정리 필요
