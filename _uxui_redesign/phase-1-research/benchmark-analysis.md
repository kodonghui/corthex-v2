## Cross-Site Pattern Synthesis

### TOP 5 Patterns to Adopt

#### 1. Dark Theme as Default (9/13 sites)
Linear, Supabase, Retool, Neon all use near-black (#09090b ~ #0D0D0D) backgrounds.
**Evidence**: 01-linear, 05-supabase, 08-retool, 14-neon screenshots.
**Adopt for**: "command" theme. Matches CORTHEX's existing dark themes (imperial, tactical, mystic, stealth).
**Implementation**: CSS `--color-bg: #09090b` for dark themes.

#### 2. Collapsible Sidebar + Content Split (10/13 apps)
Linear, Notion, Slack, Plane, Dub all show sidebar navigation in their product demos.
**Evidence**: 01-linear, 02-notion, 04-slack, 09-plane, 15-dub product screenshots.
**Adopt for**: CORTHEX already has this. Validate with 280px sidebar width, collapsible to 64px icons.
**Implementation**: Already implemented in layout.tsx/sidebar.tsx. Just needs theme variable migration.

#### 3. Minimal Top Navigation with Mega-Menu Dropdowns (11/13 sites)
All sites use a clean top nav with button-triggered dropdown menus.
**Evidence**: All screenshots show this pattern.
**Adopt for**: Landing page. Admin console could benefit from simplified top bar.
**Implementation**: Existing topbar + breadcrumbs in app layout. Landing needs redesign.

#### 4. Two-CTA Hero Pattern (12/13 sites)
Primary CTA (filled, dark) + Secondary CTA (outlined, subtle).
**Evidence**: Vercel "Start Deploying" + "Get a Demo", Plane "Try Business" + "Talk to human", etc.
**Adopt for**: Landing page hero. Also useful for page headers with actions.
**Implementation**: Button variants: primary (filled) + secondary (outlined).

#### 5. Trust Logos + Social Proof (11/13 sites)
Company logos displayed below hero. Trust badges, customer quotes.
**Evidence**: All sites except Lemon Squeezy have logo bars.
**Adopt for**: Landing page. Build credibility section.
**Implementation**: Logo bar component with grayscale filter + hover color effect.

### TOP 3 Anti-Patterns to Avoid

#### 1. Over-Animated Heroes
Stripe's gradient mesh and Neon's light bars are impressive but heavy. >500KB extra assets.
**Evidence**: 06-stripe (512KB screenshot), 14-neon (884KB screenshot).
**Avoid**: Heavy WebGL/canvas animations on dashboard pages. Landing page only if needed.

#### 2. Single-Theme Lock-in
Most sites only offer one color mode. CORTHEX needs multi-theme switching.
**Avoid**: Hardcoding any theme. All via CSS variables + data-theme attribute.

#### 3. Playful/Casual Tone for Enterprise Tools
Lemon Squeezy's purple party vibe doesn't fit AI management platforms.
**Evidence**: 11-lemonsqueezy — too casual for daily 8-hour use dashboard.
**Avoid**: Overly bright accents, playful illustrations in app chrome. Keep for landing page only.

## Design Direction Summary for 3 Themes

### Command Theme (Dark, Power User)
**Inspired by**: Linear + Supabase + Retool
- Near-black background (#09090b)
- Monochrome with single bright accent
- Dense information display
- Command palette (Cmd+K) prominent
- Tight spacing, minimal borders

### Studio Theme (Light, Collaborative)
**Inspired by**: Notion + Plane + Cal.com + Dub
- White/light gray background
- Black text, minimal color
- Clean cards with subtle borders
- Airy spacing, generous whitespace
- Friendly but professional

### Corporate Theme (Professional, Enterprise)
**Inspired by**: Vercel + Clerk + Stripe
- White background with premium accents
- Gradient or mesh accent elements (subtle)
- Balanced density
- Strong typography hierarchy
- Trust/credibility focused
