# Part 1-5: 직원 관리 테스트 (4분)

## 성공 기준
직원 생성 + 임시 비밀번호 표시 + 검색/필터 + 비활성화

## 테스트 단계

사이드바 → 조직 관리 → 직원 관리

### 직원 생성
1. "Add Employee" 클릭
2. 아이디: `cqa01`
3. 이름: `크롬검수원`
4. 이메일: `cqa01@chrometest.dev`
5. 부서 체크박스: 아무 부서 1개 선택
6. "초대" 클릭
7. **확인**: 임시 비밀번호 모달 팝업
8. **확인**: 비밀번호가 빈칸이 아닌 실제 문자열
9. "복사" 클릭 → 복사 확인
10. **비밀번호 메모** (part1-08에서 사용)
11. "확인" 클릭 → 모달 닫기
12. **확인**: 직원 목록에 "크롬검수원" 추가됨
13. 스크린샷: `screenshots/part1-05-create.png`

### 임시 비밀번호 재확인
14. "크롬검수원" 행의 열쇠 아이콘 클릭
15. **확인**: 임시 비밀번호 모달 표시
16. **확인**: "직원이 첫 로그인하면 이 비밀번호는 사라집니다" 안내
17. "확인" 클릭

### 검색
18. 검색창에 `크롬` 입력
19. **확인**: "크롬검수원"만 필터됨

### 부서 필터
20. 부서 드롭다운 → 특정 부서 선택
21. **확인**: 해당 부서 직원만 표시

### 상태 필터
22. 상태 드롭다운 → "Active" 선택
23. **확인**: 활성 직원만 표시

### 수정
24. 직원 행의 편집 아이콘 클릭
25. 이름을 `크롬검수원수정`으로 변경
26. "저장" 클릭
27. **확인**: 변경 반영됨

### 비활성화
28. 쓰레기통 아이콘 클릭
29. **확인**: 확인 다이얼로그
30. "비활성화" 클릭
31. **확인**: 상태 "Inactive"로 변경
32. 스크린샷: `screenshots/part1-05-deactivate.png`
