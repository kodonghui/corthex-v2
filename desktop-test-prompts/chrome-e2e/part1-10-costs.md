# Part 1-10: 비용 관리 테스트 (2분)

## 성공 기준
시간 필터 전환 + 3탭 전환 + 예산 입력 + Export

## 테스트 단계

사이드바 → 운영 → 비용 관리

1. "24H" 클릭 → **확인**: 데이터 변경
2. "7D" 클릭 → **확인**: 기간 변경
3. "30D" 클릭, "ALL" 클릭 → 각각 확인
4. 탭 "부서별" 클릭 → **확인**: 부서별 비용 표시
5. 탭 "에이전트별" 클릭 → **확인**: 에이전트별 비용 표시
6. 탭 "모델별" 클릭 → **확인**: 모델별 비용 표시
7. 스크린샷: `screenshots/part1-10-costs.png`
8. 예산 설정에 `100` 입력 → "저장" 클릭
9. **확인**: 프로그레스 바 또는 예산 반영
10. "Export CSV" 클릭
11. **확인**: 파일 다운로드 시작
12. 스크린샷: `screenshots/part1-10-budget.png`
