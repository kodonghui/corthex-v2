# Part 2-07: 라이브러리 + 파일 + 기밀문서 (2분)

## 성공 기준
지식 베이스 목록 + 파일 업로드 UI + 기밀문서 접근

## 테스트 단계

사이드바 → 라이브러리 클릭

1. **확인**: 라이브러리 페이지 로딩 (폴더/문서 목록)
2. 폴더 있으면 클릭 → 하위 문서 확인
3. "새 문서" 또는 "추가" 버튼이 있으면 클릭
4. 스크린샷: `screenshots/part2-07-knowledge.png`

사이드바 → 파일 클릭

5. **확인**: 파일 관리 페이지 로딩
6. 파일 목록 확인, 업로드 버튼 확인
7. 스크린샷: `screenshots/part2-07-files.png`

사이드바 → 기밀문서 클릭

8. **확인**: 기밀문서 페이지 로딩 (권한 체크)
9. 접근 가능하면 문서 목록 확인, 불가하면 권한 에러 메시지 확인
10. 스크린샷: `screenshots/part2-07-classified.png`
