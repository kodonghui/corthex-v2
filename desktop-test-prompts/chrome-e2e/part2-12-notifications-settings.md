# Part 2-12: 알림 + 설정 (2분)

## 성공 기준
알림 목록 + 설정 탭 전환 + 수정

## 테스트 단계

사이드바 → 알림 클릭

1. **확인**: 알림 페이지 로딩
2. 알림 목록 확인 (읽음/안읽음 구분)
3. 알림 항목 클릭 → 상세 또는 해당 페이지 이동
4. "모두 읽음" 버튼 있으면 클릭
5. 스크린샷: `screenshots/part2-12-notifications.png`

사이드바 → 설정 클릭

6. **확인**: 설정 페이지 로딩
7. 탭 목록 확인 (여러 탭이 있을 수 있음)
8. 각 탭 클릭 → 내용 로딩 확인
9. 프로필/비밀번호 변경 폼 있으면 확인 (저장은 하지 말 것)
10. 테마/언어 설정 있으면 변경 테스트
11. 스크린샷: `screenshots/part2-12-settings.png`
