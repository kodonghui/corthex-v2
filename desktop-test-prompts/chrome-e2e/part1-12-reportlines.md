# Part 1-12: 보고 라인 테스트 (2분)

## 성공 기준
상사 선택 + 저장 + 새로고침 후 데이터 유지

## 테스트 단계

사이드바 → 조직 관리 → 보고 라인

1. **확인**: 직원 목록과 상사 선택 드롭다운 보이는지
2. 아무 직원의 드롭다운에서 상사 선택 → **확인**: "변경됨" 표시
3. "저장" 클릭 → **확인**: 토스트 메시지
4. **새로고침** → **확인**: 방금 저장한 상사 그대로 유지
5. 스크린샷: `screenshots/part1-12-reportlines.png`
6. 새 보고 라인 추가: Reporter + Supervisor 선택 → "추가" 클릭
7. **확인**: 새 보고 라인 행 추가됨
8. 스크린샷: `screenshots/part1-12-added.png`
