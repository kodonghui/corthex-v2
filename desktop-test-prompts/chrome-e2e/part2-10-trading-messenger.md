# Part 2-10: 전략실(Trading) + 메신저 + AGORA (2분)

## 성공 기준
전략실 차트/데이터 + 메신저 UI + AGORA 마켓플레이스 확인

## 테스트 단계

사이드바 → 전략실 클릭

1. **확인**: 전략실 페이지 로딩
2. 차트/포트폴리오/워치리스트 등 위젯 확인
3. 탭이나 타임프레임 필터 있으면 클릭
4. 스크린샷: `screenshots/part2-10-trading.png`

사이드바 → 메신저 클릭

5. **확인**: 메신저 페이지 로딩
6. 채널 목록 확인
7. 채널 클릭 → 메시지 목록 확인
8. 메시지 입력창 확인
9. 스크린샷: `screenshots/part2-10-messenger.png`

사이드바 → AGORA 클릭

10. **확인**: AGORA 페이지 로딩 (에이전트 마켓플레이스 또는 커뮤니티)
11. 카드/항목 클릭 → 상세 확인
12. 스크린샷: `screenshots/part2-10-agora.png`
