# Part 2-06: 작업(Jobs) + 워크플로우 (2분)

## 성공 기준
작업 목록 + 생성 + 워크플로우 페이지 확인

## 테스트 단계

사이드바 → 작업(Jobs) 클릭

1. **확인**: 작업 목록 페이지 로딩
2. "새 작업" 또는 "+" 버튼 클릭
3. **확인**: 작업 생성 다이얼로그/폼 표시
4. 이름: `QA테스트작업`, 설명: `크롬 E2E 테스트용`
5. 스케줄/반복 설정이 있으면 확인
6. "생성" 클릭
7. **확인**: 목록에 새 작업 추가
8. 스크린샷: `screenshots/part2-06-jobs.png`

사이드바 → 워크플로우 클릭

9. **확인**: 워크플로우 페이지 로딩 (n8n 연동 또는 자체 UI)
10. 워크플로우 목록 또는 에디터 확인
11. 스크린샷: `screenshots/part2-06-workflows.png`
