# Part 2-05: 에이전트 + 에이전트 기억 (2분)

## 성공 기준
에이전트 목록 + 상세 + 기억 대시보드 확인

## 테스트 단계

사이드바 → 에이전트 클릭

1. **확인**: 에이전트 목록 표시 (카드 또는 테이블)
2. 에이전트 하나 클릭 → 상세 패널 확인 (이름, 역할, 상태, 소울)
3. 필터/검색 있으면 테스트
4. 스크린샷: `screenshots/part2-05-agents.png`

사이드바 → 에이전트 기억 클릭

5. **확인**: 기억 대시보드 로딩
6. 기억 목록 또는 타임라인 확인
7. 기억 항목 클릭 → 상세 확인
8. 스크린샷: `screenshots/part2-05-memories.png`
