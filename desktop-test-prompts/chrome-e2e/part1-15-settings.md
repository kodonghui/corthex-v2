# Part 1-15: 설정 테스트 (1분)

## 성공 기준
3탭 전환 + 수정 가능

## 테스트 단계

사이드바 하단 → 회사 설정

1. **General 탭**: 회사명 표시 확인, 수정 가능한지 클릭
2. **API Keys 탭** 클릭: 등록된 키 목록 확인
3. **Agent Settings 탭** 클릭: Handoff Depth 슬라이더 확인
4. 슬라이더를 드래그해서 값 변경 → **확인**: 변경 반영
5. 스크린샷: `screenshots/part1-15-settings.png`

## 결과 저장
results/part1-15.md에 각 단계 PASS/FAIL 기록
