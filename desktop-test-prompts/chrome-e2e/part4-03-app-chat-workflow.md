# Part 4-03: App — 에이전트와 대화 + 업무 (5분)

## 시나리오
CEO가 에이전트와 대화하고, 작업 생성하고, 보고서를 만드는 실제 업무 플로우.

## 테스트 단계

### 1. 에이전트와 대화
1. 사이드바 → 채팅
2. "새 대화" 클릭 → 에이전트 선택: 넥스트비서
3. 메시지 입력: `안녕하세요, 이번 주 일정을 정리해주세요`
4. 전송 → **확인**: 메시지 표시
5. 30초 대기 → 에이전트 응답 확인 (응답 또는 에러 기록)
6. 스크린샷: `screenshots/part4-03-chat.png`
7. 후속 메시지: `개발본부 주간회의를 수요일 2시로 잡아주세요`
8. 전송 → 응답 확인

### 2. 작업(Job) 생성
9. 사이드바 → 작업(Jobs)
10. "새 작업" 클릭
11. 이름: `주간 보안 점검`, 설명: `매주 월요일 자동 보안 스캔`
12. 생성 → **확인**: 목록에 추가
13. 스크린샷: `screenshots/part4-03-job.png`

### 3. 보고서 생성
14. 사이드바 → 보고서
15. "새 보고서" 클릭
16. 제목: `넥스트웨이브 1주차 보고서`, 내용: `테스트 기간 동안의 활동 요약`
17. 생성 → **확인**: 보고서 목록에 추가
18. 스크린샷: `screenshots/part4-03-report.png`

### 4. 라이브러리에 문서 추가
19. 사이드바 → 라이브러리
20. 문서 추가 버튼 → 제목: `온보딩 가이드`, 내용: `넥스트웨이브테크 신입 온보딩 절차`
21. 저장 → **확인**: 목록에 추가
22. 스크린샷: `screenshots/part4-03-knowledge.png`
