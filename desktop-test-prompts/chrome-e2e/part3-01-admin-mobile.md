# Part 3-01: Admin 모바일 핵심 페이지 (3분)

## 뷰포트: 390x844 (iPhone 14)

## 테스트 단계

1. /admin/login 접속 → admin/admin1234 로그인
2. **확인**: 로그인 폼 모바일 레이아웃 (가로 스크롤 없음)
3. 스크린샷: `screenshots/part3-01-login.png`

로그인 후:

4. 대시보드 → **확인**: 카드 세로 스택, 가로 넘침 없음
5. 스크린샷: `screenshots/part3-01-dashboard.png`
6. 햄버거 메뉴(☰) 클릭 → **확인**: 사이드바 오버레이 열림
7. 스크린샷: `screenshots/part3-01-sidebar.png`
8. 회사 페이지 → **확인**: 테이블이 카드 뷰로 전환 또는 가로 스크롤
9. 직원 페이지 → **확인**: 동일
10. 에이전트 페이지 → **확인**: 동일
11. 비용 페이지 → **확인**: 차트 축소, 탭 스크롤 가능
12. 설정 페이지 → **확인**: 탭 가로 스크롤, 폼 필드 전체 너비
13. 스크린샷: `screenshots/part3-01-settings.png`
