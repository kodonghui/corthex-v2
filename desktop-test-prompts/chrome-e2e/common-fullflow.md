## 규칙
- 각 단계 실행 후 결과를 PASS/FAIL로 보고
- 스크린샷을 반드시 찍어서 screenshots/ 폴더에 저장
- **이전 파트에서 생성한 데이터가 다음 파트에서 사용됨 — 순서 필수**
- 에러 발생 시 재시도 1회, 그래도 실패하면 FAIL 기록 후 다음 진행
- 결과를 results/part4-NN.md 파일에 저장
