# Part 2-09: 마케팅 파이프라인 + 콘텐츠 승인 + SNS (2분)

## 성공 기준
마케팅 파이프라인 표시 + 승인 워크플로우 + SNS 페이지 확인

## 테스트 단계

사이드바 → 마케팅 파이프라인 클릭

1. **확인**: 파이프라인 페이지 로딩 (칸반 또는 리스트)
2. 파이프라인 단계/카드 확인
3. 스크린샷: `screenshots/part2-09-marketing.png`

사이드바 → 콘텐츠 승인 클릭

4. **확인**: 승인 대기 목록 표시
5. 승인/거부 버튼 확인
6. 스크린샷: `screenshots/part2-09-approval.png`

사이드바 → SNS 클릭

7. **확인**: SNS 관리 페이지 로딩
8. 계정 목록 또는 게시물 관리 UI 확인
9. 새 게시물 작성 버튼 있으면 클릭 → 폼 확인
10. 스크린샷: `screenshots/part2-09-sns.png`
