# Part 2-11: 통신로그 + 작전일지 + 비용 + 전력분석 (2분)

## 성공 기준
4개 모니터링 페이지 정상 로딩 + 데이터 표시

## 테스트 단계

사이드바 → 통신로그 클릭

1. **확인**: 활동 로그 페이지 로딩
2. 로그 항목 표시 확인 (날짜, 유저, 액션)
3. 필터/검색 있으면 테스트
4. 스크린샷: `screenshots/part2-11-activity.png`

사이드바 → 작전일지 클릭

5. **확인**: 작전 일지 로딩
6. 일지 항목 확인
7. 스크린샷: `screenshots/part2-11-ops.png`

사이드바 → 비용 클릭

8. **확인**: 비용 페이지 로딩
9. 비용 차트/테이블 확인
10. 시간 필터 (일/주/월) 있으면 변경 테스트
11. 스크린샷: `screenshots/part2-11-costs.png`

사이드바 → 전력분석 클릭

12. **확인**: 성능 분석 페이지 로딩
13. 차트/메트릭 확인
14. 스크린샷: `screenshots/part2-11-performance.png`
