# Part 2-03: 채팅 (3분)

## 성공 기준
새 채팅 세션 생성 + 에이전트 선택 + 메시지 전송 + 응답 수신

## 테스트 단계

사이드바 → 채팅 클릭

1. **확인**: 채팅 페이지 로딩
2. "새 대화" 또는 "+" 버튼 클릭
3. **확인**: 에이전트 선택 UI 표시
4. 에이전트 하나 선택 (비서실장 또는 아무거나)
5. **확인**: 채팅 인터페이스 표시 (입력창, 전송 버튼)
6. 스크린샷: `screenshots/part2-03-chat-new.png`
7. 메시지 입력: `안녕하세요, 테스트입니다`
8. 전송 버튼 클릭 또는 Enter
9. **확인**: 메시지가 채팅창에 표시
10. **확인**: 에이전트 응답 대기 표시 (로딩 인디케이터)
11. 30초 대기 — 응답이 오면 PASS, 에러 토스트 뜨면 내용 기록
12. 스크린샷: `screenshots/part2-03-chat-response.png`
13. 왼쪽 세션 목록에서 방금 대화 확인
14. 다른 세션 클릭 → 이전 대화 로딩 확인
