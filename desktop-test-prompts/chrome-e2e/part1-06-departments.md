# Part 1-6: 부서 관리 테스트 (2분)

## 성공 기준
부서 생성/검색/상세패널/수정/삭제

## 테스트 단계

사이드바 → 조직 관리 → 부서 관리

### 부서 생성
1. "Create Department" 클릭
2. 이름: `보안점검팀`
3. 설명: `보안 취약점 점검`
4. "생성" 클릭
5. **확인**: 목록에 "보안점검팀" 추가
6. 스크린샷: `screenshots/part1-06-create.png`

### 검색
7. 검색창에 `보안` 입력
8. **확인**: "보안점검팀"만 필터됨

### 상세 패널
9. "보안점검팀" 행 클릭
10. **확인**: 우측 상세 패널 열림 (부서명, ID, 설명, 에이전트 목록)
11. 스크린샷: `screenshots/part1-06-detail.png`

### 인라인 수정
12. 상세 패널의 편집 아이콘 클릭
13. 설명을 `보안 취약점 점검 및 감사`로 변경
14. "저장" 클릭
15. **확인**: 토스트 메시지 + 설명 변경됨

### 삭제
16. 쓰레기통 아이콘 클릭
17. **확인**: cascade 분석 모달 (소속 에이전트 수 등)
18. "강제 종료" 또는 "비활성화" 선택
19. **확인**: 부서 상태 변경
20. 스크린샷: `screenshots/part1-06-delete.png`
