# Part 2-02: 대시보드 + NEXUS (2분)

## 성공 기준
대시보드 위젯 로딩 + NEXUS 조직도 표시

## 테스트 단계

사이드바 → Dashboard 클릭

1. **확인**: 대시보드 로딩 (로딩 스피너 후 데이터 표시)
2. 위젯/카드 확인: 에이전트 현황, 비용, 최근 활동 등
3. 스크린샷: `screenshots/part2-02-dashboard.png`

사이드바 → NEXUS 클릭

4. **확인**: NEXUS 페이지 로딩
5. 조직도/워크플로우 시각화 확인
6. 노드 클릭 → 상세 정보 표시 확인
7. 스크린샷: `screenshots/part2-02-nexus.png`
