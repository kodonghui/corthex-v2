# Part 2-04: 조직 + 부서 + 티어 (2분)

## 성공 기준
조직도 표시 + 부서 목록 + 티어 구조 확인

## 테스트 단계

사이드바 → 조직 클릭

1. **확인**: 조직 페이지 로딩 (조직도 또는 직원 목록)
2. 직원 카드/행 클릭 → 상세 정보 확인
3. 스크린샷: `screenshots/part2-04-organization.png`

사이드바 → 부서 클릭

4. **확인**: 부서 목록 표시
5. 부서 하나 클릭 → 소속 직원/에이전트 표시 확인
6. 스크린샷: `screenshots/part2-04-departments.png`

사이드바 → 티어 클릭

7. **확인**: 티어 구조 표시 (등급별 권한/역할)
8. 각 티어 클릭 → 상세 확인
9. 스크린샷: `screenshots/part2-04-tiers.png`
