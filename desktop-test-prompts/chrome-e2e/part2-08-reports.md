# Part 2-08: 보고서 (2분)

## 성공 기준
보고서 목록 + 생성 + 상세 보기

## 테스트 단계

사이드바 → 보고서 클릭

1. **확인**: 보고서 목록 페이지 로딩
2. 기존 보고서 있으면 하나 클릭 → 상세 보기 확인
3. 스크린샷: `screenshots/part2-08-reports-list.png`
4. "새 보고서" 또는 "+" 버튼 클릭
5. **확인**: 보고서 생성 폼/다이얼로그 표시
6. 제목: `QA검수 보고서`, 내용: `크롬 E2E 테스트 결과 보고`
7. 에이전트/부서 선택 옵션 있으면 선택
8. "생성" 클릭
9. **확인**: 보고서 생성 성공 (목록에 추가 또는 상세 페이지로 이동)
10. 스크린샷: `screenshots/part2-08-reports-created.png`
11. 보고서 상세에서 댓글/코멘트 기능 있으면 테스트
