- 기대 결과: "순환 참조가 발생합니다" 등 사용자 친화적 에러 메시지 표시

- 콘솔 에러: 없음
- 네트워크 에러: PUT /api/admin/report-lines 400 Bad Request
- 스크린샷: ss_5775iq47f (400 에러 후 상태)
- 심각도: Major
- 추정 원인: 서버에서 순환 참조를 올바르게 감지하여 400 반환하지만, 프론트엔드에서 에러 응답의 메시지를 토스트로 보여주지 않음

### BUG-002: 저장 버튼 클릭 시 브라우저 렌더러 일시 멈춤
