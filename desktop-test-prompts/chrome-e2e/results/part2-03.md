## UX 탐색 발견사항
- **빈 메시지 전송 (버튼 클릭)** → 전송되지 않음. 정상 동작
- **빈 메시지 전송 (Enter 키)** → 전송되지 않음. 정상 동작
- **파일 첨부 버튼** → 클릭 시 OS 파일 선택 대화상자 열림. 아이콘이 클립 모양. 정상
- **WEB SEARCH / DATA ANALYSIS 배지** → 클릭 시 반응 없음. 정보 표시용 배지. 정상
- **에이전트 아바타 호버** → 특별한 툴팁 없음. 에이전트 이름이나 역할 툴팁이 있으면 UX 향상 가능
- **Delete Session 버튼** → 빨간색 텍스트로 위험 액션 표시. 호버 시 하이라이트. 별도 확인 대화상자가 있는지 미확인 (테스트용 세션 보존을 위해 클릭하지 않음)
- **세션 패널 숨김 문제** → 스크롤 후 세션 패널이 사라지는 현상 발견. scroll_to로 세션 패널 영역에 이동해야 다시 보임. 세션 패널이 고정(sticky)되지 않아 채팅 내용이 길어지면 세션 전환이 불편할 수 있음
- **"CORTHEX AI can make mistakes. Consider verifying important information."** 면책 문구가 하단에 표시 → 적절한 UX
