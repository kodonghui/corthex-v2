## UX 탐색 발견사항

### 에이전트 페이지
- 검색창 "에이전트 검색..." → "보안" 입력 시 보안감사봇만 필터링됨 (정상)
- 부서 필터 드롭다운: 전체 부서/미배속/경영지원실/보안점검팀 옵션 있음 (UI는 정상, 필터링 미동작)
- 에이전트 카드 클릭 → 하단에 상세 패널 펼쳐짐 (정상)
- 상세 패널 "개요" 탭: 최근 활동(빈 상태) + 성능 지표(모델/등급/자동학습/허용도구) 표시
- 상세 패널 "Soul" 탭: 마크다운 에디터 + 13개 템플릿 변수(agent_list, subordinate_list, tool_list 등) + 프리뷰 버튼 + A/B 성격 비교 체크박스 + Soul 저장 버튼
- 상세 패널 "작업 이력" 탭: "작업 이력이 없습니다" 빈 상태 정상 표시
- 상세 패널 "설정" 탭: 모델(claude-sonnet-4-6), 등급(전문가), 자동학습(OFF), 비서에이전트(NO), 허용도구(없음) 표시
- "수정" 버튼과 "비활성화" 버튼이 상세 패널 상단에 표시됨
- "+ 에이전트 생성" 버튼 존재 (클릭하지 않음 - 데이터 생성 방지)

### 에이전트 기억 페이지
