1. **에이전트 생성 모달 정상 동작** → /agents → "+ 에이전트 생성" 버튼 클릭 → 모달 정상 열림. 필드: 에이전트 이름*(필수), 영문 이름, 소속 부서(미배속 기본), 등급(전문가 기본), 모델명(claude-haiku-4-5 기본), 역할/전문분야, CLI 소유 인간직원(없음 기본), 비서 에이전트 토글. 취소/생성 버튼. (ss_9962xwr84에서 확인)
