### BUG-001: 예산 0원일 때 사용량 퍼센트 계산 오류
- 페이지 URL: https://corthex-hq.com/admin/costs
- 재현 단계: 1. 비용 관리 페이지 접속 2. 예산에 0 입력 3. "저장" 클릭
- 기대 결과: 예산 0원이면 사용량 퍼센트가 0% 또는 "예산 미설정" 표시

- 콘솔 에러: 없음
- 심각도: Minor
- 추정 원인: 예산이 0일 때 fallback 퍼센트(83%)가 사용되거나, 0으로 나누기 시 기본값이 노출됨
