# Part 1-4: 회사 관리 테스트 (3분)

## 성공 기준
회사 생성/수정/삭제/검색 전부 작동

## 테스트 단계

사이드바 → 조직 관리 → 회사 관리

### 회사 생성
1. "Create Company" 클릭
2. 이름: `크롬QA공사` 입력
3. 슬러그: `chrome-qa-corp` 입력
4. "생성" 클릭
5. **확인**: 목록에 추가됨
6. **확인**: 토스트 메시지
7. 스크린샷: `screenshots/part1-04-create.png`

### 검색
8. 검색창에 `크롬` 입력
9. **확인**: "크롬QA공사"만 필터됨

### 수정
10. 회사 카드의 편집 아이콘 클릭
11. 이름을 `크롬QA공사수정`으로 변경
12. "저장" 클릭
13. **확인**: 변경 반영됨
14. 스크린샷: `screenshots/part1-04-edit.png`

### 삭제 (비활성화)
15. 쓰레기통 아이콘 클릭
16. **확인**: 확인 다이얼로그 표시
17. **확인**: "비활성화 후 영구 삭제 가능" 안내 문구
18. "삭제" 클릭
19. **확인**: 회사가 비활성화됨
20. 스크린샷: `screenshots/part1-04-delete.png`

### 중복 슬러그 에러
21. "Create Company" 클릭
22. 슬러그: 기존 회사와 동일한 슬러그 입력
23. "생성" 클릭
24. **확인**: 에러 메시지 표시

## 결과 저장
results/part1-04.md에 각 단계 PASS/FAIL 기록
