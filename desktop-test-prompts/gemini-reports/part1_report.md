**테스트 대상**: Part 1 (Admin 패널)
**진행률**: 15/15 섹션 완료 (100%)

## 1. 종합 요약
- **정상 작동 기능**: 로그인 및 계정 권한 분리(CEO vs 일반), 대시보드 시각화, 직원/부서/AI 에이전트 C/R/U/D 및 권한 설정, 에이전트 세부 속성 제어(시맨틱 캐시, Handoff Depth 등), 비용 관리 차트 및 예산 업데이트, 기존 등록된 API 키/CLI 연동, 조직도(NEXUS Chart) 시각화 및 내보내기 기능.
- **버그 요약**: 총 7건 발견 (Critical 1건, Major 4건, Minor 1건, Cosmetic 1건)
- **종합 의견**: Admin 패널의 코어 플랫폼 기능(데이터 생성/조회, 권한, 시각화)은 놀랍도록 안정적으로 동작합니다. 다만 사용자 여정의 첫 단계인 '온보딩' 경험과 일부 '데이터 삭제(Delete)' 과정에서 마찰을 일으키는 버그가 발견되어 최우선 수정이 권장됩니다.

## 2. 발견된 버그 및 에러 상세 리포트

### [BUG-001] "올인원" 온보딩 템플릿 누락
- **심각도**: Major
- **페이지 URL**: `https://corthex-hq.com/admin/onboarding`
- **재현 단계**: 관리자 온보딩 접속 -> Step 2 진행
- **실제 결과**: 디자인된 템플릿 목록 중 "올인원" 템플릿이 제외되어 있어 초기 전체 셋업을 원클릭으로 진행할 수 없음. 

### [BUG-002] 직원 초대 시 임시 비밀번호 모달 미노출
- **심각도**: Critical
- **페이지 URL**: `https://corthex-hq.com/admin/onboarding`
- **재현 단계**: 온보딩 Step 4 화면 -> 필수 정보 입력 후 "초대하기" 클릭
- **실제 결과**: 버튼이 "초대 중..."으로 바뀌었다 돌아오나, 임시 비밀번호를 담은 모달 팝업이 노출되지 않아 신규 사용자의 최초 접근이 원천 차단됨. (이후 직원 관리 메뉴에서 수동으로 확인해야 함)

### [BUG-003] 회사 영구 삭제 버튼 무반응 (HTTP 400)
- **심각도**: Major
- **페이지 URL**: `https://corthex-hq.com/admin/companies`
- **재현 단계**: 특정 회사의 휴지통 버튼 클릭 -> 회사명 입력 -> "영구 삭제" 클릭
- **실제 결과**: 삭제 버튼이 활성화되어 클릭할 수는 있으나, 클릭 후 아무런 UI 변경/에러 피드백 없이 삭제가 동작하지 않음. 내부 콘솔 상 HTTP 400 에러 간헐적 발생.

### [BUG-004] 부서 상세 패널 호출 시 클릭 영역 한정적
- **심각도**: Minor
- **페이지 URL**: `https://corthex-hq.com/admin/departments`
- **재현 단계**: 부서 목록 테이블 내 임의의 빈 행 영역을 클릭
- **실제 결과**: 테이블 Row 단위 클릭 이벤트가 잡히지 않아, 사용자가 아주 정확하게 부서 '이름(Text)' 텍스트를 눌러야만 우측 상세 패널이 열림.

### [BUG-005] 에이전트 수정 직후 메인 헤더 타이틀 미갱신
- **심각도**: Cosmetic
- **페이지 URL**: `https://corthex-hq.com/admin/agents`
- **재현 단계**: 에이전트 상세 패널(Config 탭)에서 이름 변경 및 저장
- **실제 결과**: 데이터는 즉시 갱신되어 Input 필드에는 반영되나, 패널 우측 상단 메인 헤더에 출력되는 에이전트 이름은 패널을 종료하고 다시 켤 때까지 이전 세션 값을 유지함.

### [BUG-006] API Key Provider 선택지에 Anthropic 및 Serper 누락
- **심각도**: Major
- **페이지 URL**: `https://corthex-hq.com/admin/credentials`
- **재현 단계**: 운영 > CLI/API 키 > "Add API Key" 버튼 클릭 -> Provider 드롭다운 확인
- **실제 결과**: KIS, Telegram 등의 폼은 정상 노출되나 핵심 Provider인 Anthropic과 검색용 Serper가 선택 목록에 아예 제공되지 않음.

### [BUG-007] API Key 'DELETE' 이중 확인 클릭 시 즉각 미반영
- **심각도**: Major
- **페이지 URL**: `https://corthex-hq.com/admin/credentials`
- **재현 단계**: 등록된 API Key 행의 휴지통 클릭 -> 버튼이 붉은색 'DELETE'로 변경 -> 다시 클릭하여 최종 삭제 확인
- **실제 결과**: 확인 처리를 했음에도 UI 상 목록에서 즉시 지워지지 않고 한동안 상주하거나 새로고침 후에도 남아있는 동기화 딜레이 이슈 발생.
