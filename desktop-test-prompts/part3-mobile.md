### 3-0. 뷰포트 설정
1. F12 눌러서 DevTools 열기
2. 상단 "Toggle device toolbar" 아이콘 클릭 (또는 Ctrl+Shift+M)
3. 상단 기기 드롭다운에서 "iPhone 14 Pro" 선택 (또는 직접 390x844 입력)
4. **확인**: 상단에 "390 x 844" 표시됨
5. 화면이 모바일 크기로 변경됨

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 3-1. Hub 페이지 (iPhone 390x844)
#### 성공 기준: 사이드바 토글 + 입력창 + 버튼 클릭

1. `https://corthex-hq.com` 접속 (로그인 상태 유지)
2. **확인**: 사이드바가 숨겨져 있는지 (데스크톱에서는 항상 보임)
3. 좌측 상단 햄버거 메뉴 (≡) 클릭
4. **확인**: 사이드바가 슬라이드로 열림
5. 사이드바 밖 영역 클릭
6. **확인**: 사이드바가 닫힘
7. 예시 명령어 버튼 4개 확인 — 세로로 정렬되는지 또는 스크롤 가능한지
8. 예시 버튼 1개 클릭 → **확인**: 실행됨 (비서실장 응답 시작)
9. 하단 입력창에 `테스트` 입력
10. **확인**: 키보드가 올라와도 입력창이 화면에 보임 (가려지지 않음)
11. Execute 버튼 클릭 → **확인**: 전송됨

**에러 기준:**
- 전체 페이지가 가로 스크롤되면 → **버그** (가로 스크롤은 테이블에서만 허용)
- 버튼/링크가 너무 작아서 터치 불가 (44px 미만) → **버그**

### 3-2. Chat 페이지 (iPhone 390x844)
#### 성공 기준: 세션 목록 → 대화 → 뒤로가기

1. 햄버거 메뉴 → 채팅 클릭
2. **확인**: 세션 목록이 표시됨
3. 세션 1개 클릭 → **확인**: 대화 화면으로 전환
4. 메시지 입력 → 전송 → **확인**: 응답 수신
5. 뒤로가기 버튼 (←) 클릭 → **확인**: 세션 목록으로 복귀
6. **확인**: 메시지 버블이 화면 너비에 맞게 줄 바꿈됨 (넘치지 않음)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 3-3. Agents 페이지 (iPhone 390x844)
#### 성공 기준: 카드 1열 + 클릭 + 스크롤

1. 햄버거 메뉴 → 에이전트 클릭
2. **확인**: 에이전트 카드가 1열로 세로 정렬됨 (2열이면 확인: 잘리지 않는지)
3. 카드 1개 클릭 → **확인**: 상세 정보 열림
4. 아래로 스크롤 → **확인**: 부드럽게 스크롤됨

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 3-4. Admin 사이드바 (iPhone 390x844)
#### 성공 기준: 햄버거 메뉴 + 테이블 가로 스크롤

1. `https://corthex-hq.com/admin` 접속
2. **확인**: 사이드바가 숨겨져 있음
3. 햄버거 메뉴 클릭 → **확인**: 사이드바 열림
4. 메뉴 항목 "직원 관리" 클릭 → **확인**: 페이지 이동 + 사이드바 닫힘
5. 직원 테이블 확인:
   - **확인**: 테이블이 가로 스크롤 가능 (전체 페이지가 아닌 테이블만)
   - **확인**: 텍스트가 잘리지 않거나, 잘려도 말줄임표(...) 처리

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 3-5. Admin 주요 페이지 (iPhone 390x844)
#### 성공 기준: 대시보드/에이전트/비용 페이지 모바일에서 정상 로드

1. 대시보드 → **확인**: 카드 3개가 세로 정렬 (1열)
2. 에이전트 페이지 → **확인**: 테이블 가로 스크롤 가능
3. 비용 페이지 → **확인**: 차트가 화면 너비에 맞게 리사이즈

### 3-6. iPad 테스트 (768x1024)
#### 성공 기준: 태블릿 레이아웃 정상

1. DevTools 기기 변경: "iPad" 선택 (768x1024)
2. Hub 페이지 → **확인**: 사이드바가 항상 보이는지 또는 토글인지
3. 에이전트 페이지 → **확인**: 카드 2열 정렬
4. 직원 테이블 → **확인**: 컬럼이 잘리지 않고 전부 보임
