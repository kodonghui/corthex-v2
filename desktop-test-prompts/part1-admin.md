   - "이메일" 필드: `emp1@test.com` 입력

3. "초대" 버튼 클릭
4. 1~3초 대기 (서버 응답)
5. **확인**: 임시 비밀번호 모달이 뜸
6. **확인**: `<code>` 태그 안에 비밀번호가 **빈칸이 아닌** 실제 문자열로 표시됨
7. "복사" 버튼 클릭하여 복사 확인
8. 비밀번호를 기억/메모 (나중에 1-7b에서 사용)
9. "확인" 클릭하여 모달 닫기
10. **확인**: 직원 목록 테이블에 "테스트직원"이 추가됨

**1-5-2. 임시 비밀번호 재확인 테스트**
11. 방금 만든 "테스트직원" 행의 열쇠(KeyRound) 아이콘 클릭
12. **확인**: 임시 비밀번호 모달이 바로 뜨고, 비밀번호가 표시됨 (리셋 확인 없이)
13. **확인**: 모달 안내 문구 "직원이 첫 로그인하면 이 비밀번호는 사라집니다" 표시
14. "확인" 클릭하여 모달 닫기

**1-5-3. 검색**
15. 검색창에 `테스트` 입력
16. **확인**: "테스트직원" 행만 필터됨

**1-5-4. 부서 필터**
17. 부서 드롭다운 클릭 → 특정 부서 선택
18. **확인**: 해당 부서 직원만 표시

**1-5-5. 상태 필터**
19. 상태 드롭다운 → "Active" 선택
20. **확인**: 활성 직원만 표시
21. "Inactive" 선택 → **확인**: 비활성 직원만 표시 (없으면 빈 테이블)

22. 직원 행의 연필(편집) 아이콘 클릭
23. 이름을 "수정된직원"으로 변경
24. "저장" 클릭
25. **확인**: 목록에 "수정된직원" 반영

**1-5-7. 비밀번호 초기화**
26. 직원 행의 열쇠(KeyRound) 아이콘 클릭
27. **확인**: 비밀번호가 모달에 표시됨 (다크 테마 다이얼로그)
28. (만약 이미 첫 로그인 후면 "초기화" 확인 다이얼로그 → "초기화" 클릭)

30. 직원 행의 쓰레기통 아이콘 클릭
31. **확인**: "비활성화" 확인 다이얼로그 (다크 테마)
32. "비활성화" 클릭
33. **확인**: 직원 상태가 "Inactive"로 변경
