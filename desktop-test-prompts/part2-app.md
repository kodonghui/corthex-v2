### 2-2. Hub — 명령 센터 (5분) ★★★★★ 최핵심
이것이 CORTHEX의 핵심 기능입니다!

**UI 확인:**
1. 좌측: 사이드바 (Workspace Hub, Active Tasks, Agent Directory)
2. 중앙: 채팅/명령 영역
3. 하단: 터미널 입력창 (CORTHEX_TERMINAL_V2)
4. 예시 명령어 버튼 4개 확인

**비서실장에게 명령 (자연어):**
5. 입력창에 입력: "우리 회사 전 부서 현황 보고해줘"
6. Enter 또는 Execute 클릭
7. "명령 접수됨..." → 비서실장이 처리 → 응답 스트리밍 확인
8. 응답이 올 때까지 기다리기 (30초~2분)
9. 응답 품질 확인: 부서별 현황이 정리되어 나오는지

**@멘션 직접 지시:**
10. 입력: "@마케팅팀장 인스타그램 콘텐츠 3개 써줘"
11. 마케팅팀장 에이전트가 직접 응답하는지 확인

**슬래시 명령어:**
12. 입력: "/명령어" → 사용 가능한 명령어 목록 표시
13. 입력: "/도구점검" → 도구 상태 확인 결과

**프리셋 명령어:**
14. 예시 버튼 중 하나 클릭 → 자동 입력 + 실행

### 2-3. 채팅 — 개별 에이전트 대화 (5분) ★★★★ 핵심
사이드바 → 채팅 (또는 /chat)

**세션 생성:**
1. "+" 또는 "새 대화" 클릭
2. 에이전트 선택 모달에서 "비서실장" 선택
3. 대화 시작

**실제 대화:**
4. 입력: "오늘의 주요 이슈 3가지를 브리핑해줘"
5. 응답 대기 → 품질 확인 (마크다운 포맷, 내용의 구체성)
6. 후속 질문: "그 중 첫 번째에 대해 자세히 분석해줘"
7. 대화 컨텍스트 유지되는지 확인

**다른 에이전트:**
8. 새 세션 → "개발팀장" 선택
9. 입력: "기술 부채 분석 보고서 써줘"
10. 응답 확인

**세션 관리:**
11. 세션 목록에서 이전 세션 클릭 → 대화 이력 로드
12. 세션 이름 변경 가능한지 확인
