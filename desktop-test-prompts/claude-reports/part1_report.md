- **재현 단계**:

  2. 에이전트 행의 "EDIT" 클릭하여 상세 패널 열기
  3. Config 탭 확인
- **기대 결과**: 오프라인 에이전트의 Config 탭에 "Go Online" 초록색 버튼이 표시되어야 함

- **에러 메시지**: 없음
- **콘솔 에러**: 없음
- **재현율**: 항상
- **영향 범위**: 1-7-1b Go Online 테스트 불가

### BUG-002: Reactivate Agent 버튼 없음

- **심각도**: Major (비활성화된 에이전트 재활성화 불가)
- **페이지 URL**: https://corthex-hq.com/admin/agents
- **재현 단계**:
  1. 에이전트를 비활성화 (Deactivate Agent 클릭)
  2. 비활성화된 에이전트 [OFF]의 EDIT 클릭
  3. Config 탭 확인
- **기대 결과**: "Reactivate Agent" 초록색 버튼이 표시되어야 함
- **실제 결과**: "ALREADY DEACTIVATED" (클릭 불가 상태 텍스트)와 "PERMANENTLY DELETE" 버튼만 표시. 재활성화 방법 없음.
- **에러 메시지**: 없음
- **콘솔 에러**: 없음
- **재현율**: 항상
- **영향 범위**: 1-7-8b Reactivate 테스트 불가. 비활성화한 에이전트를 되돌릴 수 없음 (영구 삭제만 가능)

### BUG-003: 온보딩 "Next" 버튼 일반 클릭 미작동

- **심각도**: Minor (JavaScript 이벤트 디스패치로 우회 가능)

- **실제 결과**: 아무 반응 없음. JavaScript `dispatchEvent(new MouseEvent('click'))` 으로만 작동.
- **에러 메시지**: 없음
- **콘솔 에러**: 없음 (Chrome 확장 관련 에러만 있음)
- **재현율**: 항상 (브라우저 자동화 환경 한정일 수 있음)
- **영향 범위**: 온보딩 전체 플로우의 Next/Continue 버튼 동일 증상
