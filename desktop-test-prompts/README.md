# CORTHEX Desktop App 테스트 프롬프트

4개 파트로 나뉘어 있습니다. **순서대로 하나씩** Desktop App에 붙여넣으세요.

| 파트 | 파일 | 소요 시간 | 내용 |
|------|------|-----------|------|
| 1 | part1-admin.md | ~20분 | Admin 패널 전체 (로그인, 온보딩, CRUD, 에이전트, 도구 등) |
| 2 | part2-app.md | ~25분 | CEO App 전체 (Hub, 채팅, 슬래시 명령어, 협업, Big Five 등) |
| 3 | part3-mobile.md | ~3분 | 모바일 반응형 테스트 |
| 4 | part4-evaluation.md | ~5분 | 종합 평가표 작성 |

## Desktop App에 처음 줄 메시지

아래를 **첫 번째 메시지**로 보내세요:

```
CORTHEX v2 AI 플랫폼 전체 기능 테스트를 합니다.
총 4파트로 나눠서 진행할 거예요. 제가 파트별로 지시를 줄게요.
Chrome 브라우저를 열어서 실제로 모든 기능을 하나하나 사용해보세요.

기본 정보:
- Admin 패널: https://corthex-hq.com/admin (admin / admin1234)
- CEO App: https://corthex-hq.com (ceo / ceo1234)
- 에이전트 채팅은 응답이 30초~2분 걸리니 기다려주세요
- 에러가 나면 화면 캡처하고 다음으로 넘어가세요
- 각 기능 테스트 후 결과를 간단히 말해주세요 (작동/안됨/에러)

준비되면 "준비됐어"라고 말해주세요. 그러면 Part 1을 보내드리겠습니다.
```

그 다음 "준비됐어" 응답이 오면 part1-admin.md 내용을 붙여넣으세요.
Part 1 끝나면 part2-app.md, 그 다음 part3, part4 순서로.

## 중요: 파트 간 의존성

Part 1 (Admin)이 기반입니다. 여기서 회사/부서/에이전트를 만들어야 Part 2 (App)에서 쓸 수 있습니다.
만약 Part 1에서 뭔가 막히면 → Part 2로 넘어가지 말고 → 막힌 지점을 보고서에 기록하세요.
